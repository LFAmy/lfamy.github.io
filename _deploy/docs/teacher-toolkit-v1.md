## 工具 7：家長溝通話術庫

### 課後報喜（標準版）
```
{家長稱呼}你好！今日{學生名}上堂好投入呀 😊
佢喺{課題}方面進步咗好多，特別係{具體亮點}。
之前成日錯嘅{陷阱}，今日佢自己識得避開喇！👍
下一步會重點幫佢突破{下個目標}。
你有冇咩想特別同我講嘅？
```

### 課後提醒（學生不專心時）
```
{家長稱呼}你好！今日{學生名}上堂有啲分心，
可能係{原因猜測：太攰/學校活動/心情}嘅關係。
佢喺{課題}方面仲需要多啲練習。
我建議呢個星期可以同佢一齊做{具體行動}，
下次上堂我會特別留意呢方面。
有咩需要我配合嘅隨時話我知！😊
```

### 月度總結
```
{家長稱呼}你好！{學生名}{月份}學習總結：

✅ 完成：{X}堂，練習{X}題
✅ 進步：{具體進步1}、{具體進步2}
✅ 亮點：{具體亮點}

🎯 下月重點：{目標1}、{目標2}
📝 建議：{具體建議}

💰 學費到期日：{日期}
有任何問題隨時同我講！😊
```

### 考試前動員
```
{家長稱呼}你好！呈分試就到喇！📚
{學生名}呢排嘅準備情況：
✅ 已掌握：{強項課題}
⚠️ 要加強：{弱項課題}

建議呢個禮拜集中做：
1. {具體行動1}
2. {具體行動2}

我哋已經喺堂上做咗{X}次模擬，佢嘅信心同狀態都唔錯！
有咩需要隨時搵我，一齊幫{學生名}衝刺！💪
```

### 續費提醒
```
{家長稱呼}你好！{學生名}嘅{月份}學費將於{日期}到期。
呢個月佢嘅進步好明顯，特別係{具體進步}。
如果想繼續，可以喺呢度處理：{連結}
如果有任何考慮，都可以同我傾傾！😊
```

### 處理投訴
```
步驟：
1. 即時回覆：「多謝你嘅反映，我完全明白你嘅擔心。」
2. 收集資料：「可唔可以講多少少{具體情況}？」
3. 不推卸：「我會即刻了解情況同跟進。」
4. 給時間線：「我會喺24小時內畀一個完整回覆你。」
5. 上報主管：記錄→通知→跟進

❌ 不要：
- 即時辯解
- 推卸責任給學生
- 承諾無法保證的結果
```

## 工具 8：課堂時間控制速查卡

```
⏱ 霖楓65分鐘時間檢查點
┌─────────────────────────────────────┐
│ 0:00  開始！破冰                   │
│ 3:00  ⚠️ 必須進入回顧環節          │
│ 8:00  ⚠️ 必須開始核心講解          │
│ 20:00 ⚠️ 必須完成核心講解          │
│ 25:00 ⚠️ 必須完成例題示範          │
│ 40:00 ⚠️ 必須完成互動練習          │
│ 50:00 ⚠️ 必須完成陷阱揭示          │
│ 60:00 ⚠️ 必須完成獨立練習          │
│ 65:00 結束！總結+金句              │
└─────────────────────────────────────┘

時間不夠時的取捨順序：
1. 先砍「互動練習」中的次要題目
2. 再砍「獨立練習」中的挑戰題
3. 絕不砍「陷阱揭示」和「總結」

時間太多時的補充順序：
1. 加插挑戰題
2. 讓學生互相出題
3. 玩數學遊戲
```
