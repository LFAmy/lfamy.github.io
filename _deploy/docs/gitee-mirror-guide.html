Gitee 鏡像部署指南

5分鐘 · 內地訪問速度 8秒 → 0.5秒

為什麼需要 Gitee 鏡像？

✅ GitHub Pages (www.lfacademyhk.com) 在中國內地載入需 5-15 秒
✅ Gitee Pages 在中國內地載入 < 1 秒
✅ 雙平台自動同步：GitHub 推送 → Gitee 自動鏡像
✅ 香港用戶繼續用 GitHub Pages，內地用戶自動導向 Gitee

部署步驟

1

註冊 Gitee 帳號

打開 gitee.com → 註冊（可用微信登入）

2

導入 GitHub 倉庫

Gitee 首頁 → 點擊「從 GitHub 導入」→ 輸入倉庫 URL：https://github.com/LFAmy/www.lfacademyhk.com

3

開啟 Gitee Pages

進入倉庫 → 服務 → Gitee Pages → 部署目錄選 docs → 啟動

你的國內網址將會是：https://lfamy.gitee.io

4

設定自動同步

在 GitHub 倉庫新增 GitHub Actions workflow：

.github/workflows/gitee-sync.yml

name: Sync to Gitee
on:
  push:
    branches: [main]
jobs:
  sync:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
        with:
          fetch-depth: 0
      - name: Push to Gitee
        run: |
          git remote add gitee https://${GITEE_USER}:${GITEE_TOKEN}@gitee.com/lfamy/lfamy.git
          git push gitee main --force
        env:
          GITEE_USER: ${{ secrets.GITEE_USER }}
          GITEE_TOKEN: ${{ secrets.GITEE_TOKEN }}

每次推送 GitHub，自動同步到 Gitee。

5

🇭🇰 香港用戶 → www.lfacademyhk.com → Google Fonts + WhatsApp (完整體驗)
