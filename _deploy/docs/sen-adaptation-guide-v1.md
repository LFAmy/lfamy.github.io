# 霖楓學苑 · 特殊學習需要(SEN)適應指南 v1.0
## Lam Fung Academy · SEN Adaptation Guide
### ADHD · 讀寫障礙 · 數學焦慮 · 自閉譜系 · 資優
### 更新：2026-05-29

---

## 概述

本指南幫助教師識別和適應常見的特殊學習需要(SEN)學生。霖楓的「一對三」小班模式天然適合 SEN 學生，但仍需針對性策略。

---

## 一、五種常見 SEN 類型速查

| 類型 | 核心特徵 | 課堂影響 | 黃金策略 |
|------|---------|---------|---------|
| **ADHD** | 注意力短暫、坐不定、衝動 | 無法專注超過5分鐘 | 頻繁轉換活動 + 動手操作 |
| **讀寫障礙** | 閱讀困難、抄寫出錯 | 讀題慢、抄錯數字 | 口頭讀題 + 放大字體 |
| **數學焦慮** | 對數學恐懼、自我否定 | 不敢嘗試、空白 | 降低難度入口 + 大量鼓勵 |
| **自閉譜系** | 社交困難、固執、感官敏感 | 不願互動、執著細節 | 結構化流程 + 預告變化 |
| **資優** | 學習快、求知慾強但易悶 | 課堂無聊、挑戰權威 | 加速 + 深化 + 自主探索 |

## 二、ADHD 適應策略

### 課堂環境
- 座位：最靠近老師，遠離窗戶/門口（減少分心）
- 移除桌面上不必要的物品
- 使用視覺計時器（讓學生看到還剩多少時間）

### 教學策略
| 策略 | 操作 |
|------|------|
| **5分鐘轉換** | 每5分鐘轉換一次活動類型（聽→寫→說→動手） |
| **動手操作** | 用實物（積木/計數器/摺紙）教數學概念 |
| **先動後靜** | 課堂開始時做30秒伸展/深呼吸 |
| **分段任務** | 將練習分成3-4個小段，每段完成後給予即時回饋 |
| **陷阱遊戲化** | 「捉錯處」比賽，讓 ADHD 學生的能量變成優勢 |
| **獎勵即時化** | 每完成一個小目標立即給予口頭讚賞 |

### 溝通技巧
- 說話簡短清晰（不超過2句話）
- 叫名字後才說指令
- 用「做...」而非「不要...」（正向指令）

## 三、讀寫障礙適應策略

### 教材調整
- 字體放大（最小14pt，理想18pt）
- 使用非純白背景（米色/淺黃減少視覺壓力）
- 每行不超過15個中文字
- 關鍵字用粗體/顏色標記

### 教學策略
| 策略 | 操作 |
|------|------|
| **口頭讀題** | 老師讀出題目，學生只需聽和答 |
| **語音輸入** | 讓學生用說話代替書寫（錄音/口答） |
| **顏色標記** | 用不同顏色標記：已知條件(藍)、所求(紅)、公式(綠) |
| **圖像輔助** | 盡量用圖表代替純文字 |
| **步驟框架** | 提供預先寫好的解題步驟框架，學生只需填數字 |
| **減少抄寫** | 提供預印練習卷，學生只需填答案 |

### 常見錯誤模式
- 6 和 9 混淆 → 在數字下加底線標示方向
- b 和 d 混淆 → 用顏色區分
- 抄錯題目 → 用尺子逐行對齊

## 四、數學焦慮適應策略

### 識別信號
- 看到數學題就說「我唔識」
- 未看題目就放棄
- 身體緊張（握拳/咬唇/出汗）
- 說「我好蠢」、「數學好難」

### 教學策略
| 策略 | 操作 |
|------|------|
| **超低門檻** | 第一題必須是100%能做對的（建立成功感） |
| **「我細個都錯過」** | 分享老師自己的失敗經歷，正常化錯誤 |
| **過程讚賞** | 讚「你寫步驟好有條理」而非「你好聰明」 |
| **錯誤慶祝** | 「好嘢！我哋搵到一個陷阱！學到嘢喇！」 |
| **小目標** | 今天只需要完成3題。完成後就慶祝！ |
| **呼吸練習** | 開始前做3次深呼吸：「吸入信心，呼出擔心」 |

### 禁忌
- ❌ 說「咁簡單都唔識？」
- ❌ 公開比較學生
- ❌ 催逼速度
- ❌ 跳過讚賞直接指錯

## 五、自閉譜系適應策略

### 課堂環境
- 固定座位（每次一樣的位置）
- 使用視覺流程表（讓學生知道接下來做什麼）
- 避免突然的大聲或強光

### 教學策略
| 策略 | 操作 |
|------|------|
| **結構化流程** | 「首先→然後→最後」的清晰步驟 |
| **預告變化** | 提前5分鐘預告：「仲有5分鐘就會轉下一個活動」 |
| **具體語言** | 避免比喻/反諷。說「寫低答案」而非「你識做啦可？」 |
| **特殊興趣連結** | 如果學生喜歡火車→用火車做應用題例子 |
| **感官休息** | 允許學生在需要時短暫離開/閉眼休息1分鐘 |
| **視覺提示卡** | 準備「我需要幫助」「我需要休息」的卡片 |

### 社交互動調整
- 不強迫眼神接觸
- 輪流答題時給予額外等待時間
- 一對三時安排在最側邊（減少社交壓力）

## 六、資優學生適應策略

### 識別信號
- 比其他學生快2-3倍完成練習
- 喜歡問「如果...會怎樣？」
- 對重複練習感到無聊/煩躁
- 可能挑戰老師的解法

### 教學策略
| 策略 | 操作 |
|------|------|
| **壓縮課程** | 如果已掌握→跳過基礎練習，直接給挑戰題 |
| **深度代替廣度** | 不是給更多題，而是給更深層次的問題 |
| **反向問題** | 「如果答案是24，題目可能是什麼？」 |
| **小老師角色** | 讓資優生教其他學生（鞏固自己的理解） |
| **開放式挑戰** | 「有多少種方法可以解這題？」 |
| **真實應用** | 連結數學到現實問題（計算折扣、設計房間） |

## 七、一對三混合 SEN 管理

當一班中有不同 SEN 學生時的調配策略：

| 學生組合 | 座位安排 | 策略 |
|---------|---------|------|
| ADHD + 焦慮 + 一般 | ADHD最近老師，焦慮在中間 | 先給焦慮生簡單題建立信心，同時讓ADHD生動手操作 |
| 資優 + 讀寫 + 一般 | 資優在側邊，讀寫最近老師 | 資優做挑戰題，老師幫讀寫讀題 |
| 自閉 + ADHD + 焦慮 | 自閉最側邊，ADHD最近，焦慮中間 | 結構化流程給自閉，頻繁轉換給ADHD，大量鼓勵給焦慮 |

### 通用原則
1. **公平 ≠ 相同：** 每個學生需要不同的支援
2. **提前準備：** 為每個 SEN 學生準備分層材料
3. **記錄追蹤：** 每堂記錄 SEN 學生的特殊反應和有效策略
4. **家長溝通：** 定期更新 SEN 學生的適應策略效果

---

## 八、SEN 學生家長溝通

### 溝通原則
- 先分享進步和亮點
- 用具體例子說明
- 提供家居配合建議
- 定期檢討適應策略

### 溝通模板
```
{家長稱呼}你好！今日{學生名}喺{具體課題}有進步。
我哋用咗{具體策略}，佢嘅反應係{具體反應}。
建議家居可以{具體建議}。
我哋會繼續觀察同調整。有咩需要配合嘅隨時話我知！😊
```

## 九、緊急處理

| 情境 | 行動 |
|------|------|
| 學生情緒爆發 | 暫停教學→單獨冷靜→不當眾處理 |
| 感官過載（自閉） | 允許離開→提供安靜空間→降低刺激 |
| ADHD 極度分心 | 給一個「特殊任務」（幫老師拿東西/擦白板）→重新定向 |
| 焦慮發作 | 停止數學→深呼吸→轉移話題→降低難度 |
