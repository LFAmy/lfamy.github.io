# 霖楓學苑 · 教師速查手冊

## 🎯 核心原則

> **不教數學，教避開陷阱。**
> 你的價值不是「解釋數學概念」，而是「幫助學生識別和避開重複錯誤的陷阱」。

---

## 🔟 十大陷阱一覽

| # | 陷阱名稱 | 典型錯誤 | 破解口訣 |
|---|---------|---------|---------|
| T1 | 進退位 | 38+27=55 (忘進位) | 「夠10就進1，唔夠就借1」 |
| T2 | 單位換算 | 1m²=100cm² | 「長度×10，面積×100，體積×1000」 |
| T3 | 運算順序 | 6+4×2=20 | 「先乘除，後加減，括號最優先」 |
| T4 | 漏寫0 | 408÷2=24 | 「除數中間有0，商嘅中間都要有0」 |
| T5 | 分數陷阱 | 1/2+1/3=2/5 | 「分母唔同要先通分」 |
| T6 | 百分數 | +20%-20%= | 「每次變化用新基數」 |
| T7 | 應用題 | 問A答B | 「圈關鍵字→畫圖→列式→檢查單位」 |
| T8 | 圖形陷阱 | 面積當周界 | 「面積=底×高，周界=邊長加晒」 |
| T9 | 時間換算 | 1.5hr=150min | 「1小時=60分鐘，用×60唔係×100」 |
| T10 | 餘數陷阱 | 除不盡忘記餘數 | 「除到盡或寫餘數，唔好就咁停」 |

## ⏱ 60分鐘課堂 4P 流程

### P1: Probe 診斷 (0:00-3:00)
- [ ] 打招呼 + 自我介紹
- [ ] 回顧上次重點
- [ ] 每人 30 秒分享
- [ ] 預告今日目標 + 陷阱

### P2: Present 講解 (3:00-25:00)
- [ ] 展示知識點公式 (5min)
- [ ] 生活化例子 1 (5min)
- [ ] 展示陷阱 + 口訣 (7min)
- [ ] 生活化例子 2 (5min)

### P3: Practice 練習 (25:00-53:00)
- [ ] 教師示範 1 題 (5min)
- [ ] 引導練習：每人 1 題 (8min)
- [ ] 獨立練習：每人 3 題 (10min)
- [ ] 挑戰題給快者 (5min)

### P4: Post 總結 (53:00-60:00)
- [ ] 每人總結今日所學 (3min)
- [ ] 家課佈置 (3min)
- [ ] 預告下一堂 (1min)

## 👥 1對3 黃金法則

1. **座位安排：** 最弱坐中間
2. **關注分配：** 60%最弱 → 20%中間 → 20%最強
3. **點名節奏：** 每 2 分鐘輪換，不連續點同一人
4. **「我先你後」法：** 教師示範 → 中等嘗試 → 最弱練習 → 最強挑戰
5. **競爭+合作：** 前半堂計分 → 後半堂合作

## 📱 家長溝通 SOP

### 課前（上課前 1 天）
發送課前預告：時間、課題、陷阱類型、準備事項

### 課後（下課 30 分鐘內）
發送課後總結：完成內容、亮點讚賞、家課、下次上課時間

### 每月（月初）
發送月度回顧：練習統計、進步亮點、下月重點、續費提醒

### 關鍵原則
- 永遠「先讚後建議」
- 用具體例子，不用抽象評語
- 回覆時限：2 小時內
- 投訴：記錄 → 上報 → 24hr 回覆
