# 霖楓學苑 · Web App 設計方案 v1.0
## 「一對三補習的最強後盾」

---

## 一、定位與目標

### Web App 不是取代老師，而是放大老師的效能

```
傳統模式：老師 → 講義PDF → 學生
Web App：老師 → 講義(PWA) → 學生 + 自動練習 + AI診斷 + 家長儀表板
```

### 核心功能三層架構

| 層級 | 用戶 | 核心功能 |
|------|------|---------|
| 🟢 學生層 | 學生 | 講義查看、互動練習、即時回饋、進度追蹤 |
| 🟡 老師層 | 老師 | 課堂管理、學生弱項儀表板、自動出題 |
| 🔵 家長層 | 家長 | 學習報告、付費管理、直接溝通 |

## 二、技術架構

### 推薦方案：靜態網站 + PWA + Firebase

```
霖楓Web App 架構：
┌─────────────────────────────────────┐
│          前端 (PWA)                  │
│  React + Vite + Tailwind CSS        │
│  - 響應式設計（手機/平板/電腦）     │
│  - 離線可用（Service Worker）       │
│  - 可安裝到主螢幕（PWA）            │
├─────────────────────────────────────┤
│          後端 (輕量)                 │
│  Firebase / Supabase                │
│  - 學生帳號管理                     │
│  - 練習數據儲存                     │
│  - 進度追蹤                         │
├─────────────────────────────────────┤
│          內容層                      │
│  GitHub Pages (現有)                │
│  - 講義HTML/CSS                     │
│  - 題庫JSON                         │
│  - 靜態資源                         │
└─────────────────────────────────────┘
```

### 為什麼選擇這個架構？
- ✅ 霖楓已有GitHub Pages → 零額外託管成本
- ✅ PWA可離線使用 → 學生網絡不穩也能用
- ✅ Firebase免費額度足夠167學生使用
- ✅ 維護成本極低

## 三、核心功能詳解

### 3.1 AI免費診斷系統（獲客引擎）

**功能：** 家長/學生在網站上做10題診斷測驗，即時獲得弱項分析報告

```
診斷流程：
1. 選擇年級（P3-P6）
2. 做10題跨課題診斷題（5分鐘）
3. AI分析：精確定位3個最弱課題
4. 生成個人化報告 → 引導預約免費試堂
5. 收集聯絡資料 → 加入獲客漏斗
```

**題目設計原則：**
- 每題對應一個陷阱類型
- 10題涵蓋該年級5大核心課題
- 每題2分鐘限時（模擬考試壓力）
- 錯誤答案的錯法能揭示學生思維模式

### 3.2 學生練習平台

**功能：** 課後練習 + 智能出題 + 即時回饋

| 功能 | 說明 |
|------|------|
| 課後練習包 | 每堂課後自動推送10-15題針對性練習 |
| 智能出題 | 根據學生弱項，自動生成更多同類型題目 |
| 即時批改 | 做完即顯示對錯 + 陷阱提示 |
| 錯題本 | 自動收集錯題，一鍵生成複習卷 |
| 進度條 | 視覺化顯示各課題掌握度 |
| 連續打卡 | 連續練習天數追蹤，激勵持續學習 |

### 3.3 家長儀表板

**功能：** 家長隨時查看子女學習狀況

```
家長儀表板顯示：
┌──────────────────────────────┐
│  📊 小明 · P5 數學           │
│  ─────────────────────────── │
│  🏆 本週練習：4/5 天         │
│  📈 整體掌握度：78% → 85%    │
│  ⚠️ 最弱課題：異分母分數加法  │
│  🎯 下次課堂重點：方程應用題  │
│  📅 下堂：5月30日(五) 5:00pm │
│  💬 老師留言：「小明加油！」  │
└──────────────────────────────┘
```

### 3.4 老師操作台

**功能：** 課前準備 + 課中輔助 + 課後跟進

| 階段 | 功能 |
|------|------|
| 課前 | 查看學生上次弱項、準備針對性題目、發送預習提醒 |
| 課中 | 即時計分板、隨機點名器、計時器 |
| 課後 | 一鍵生成課堂報告、推送課後練習、記錄學生表現 |

## 六、技術實現建議

### 前端技術棧
- **框架：** React 18 + TypeScript
- **樣式：** Tailwind CSS（快速開發）
- **PWA：** Vite PWA Plugin
- **圖表：** Chart.js（進度圖）
- **數學渲染：** KaTeX（比MathJax更快更輕）

### 數據存儲
- **用戶認證：** Firebase Auth（Google登入）
- **資料庫：** Firebase Firestore
- **靜態資源：** GitHub Pages

### 部署
- **CI/CD：** GitHub Actions → 自動部署到GitHub Pages
- **域名：** lfamy.github.io（現有）→ 可升級為自訂域名
