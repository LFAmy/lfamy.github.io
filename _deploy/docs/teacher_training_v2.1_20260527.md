## Day 1：基本功 — 香港數學教學的DNA（1.5小時）

### Section 1.1：香港小學數學全景圖（20分鐘）

**新增✨ SSPA數據支撐表：**

| 呈分試表現 | 對應Banding | 可升中學例子 |
|-----------|------------|------------|
| 數學A級（90+） | Band 1A | 皇仁、喇沙、拔萃、聖保羅 |
| 數學B級（80-89） | Band 1B-C | 華仁、英皇、圣若瑟 |
| 數學C級（70-79） | Band 2A-B | 地區英中 |
| 數學D級（60-69） | Band 2C-3A | 中中為主 |
| 數學E級（<60） | Band 3 | 有限選擇 |

> 🔑 關鍵訊息：數學科在呈分試佔比重約25%（中英數常），數學1題≈改變Banding！

**真實升中案例（加入培訓）：**
- 案例A：P5數學75分→補習後P6呈分試92分→成功升讀Band 1A
- 案例B：P5數學88分→冇補習→P6呈分試68分→跌至Band 2C
- 案例C：P6數學62分→SSPA急救包→呈分試81分→升讀心儀英中

### Section 1.2：香港用字與解題方法（25分鐘）

**新增✨ 沉浸式改錯練習（8分鐘）：**

操作流程：
1. 播放一段3分鐘「內地導師教香港數學」模擬影片（培訓師扮演）
2. 影片中故意植入6-8個不符合香港教法的錯誤
3. 新老師拿「改錯紙」邊看邊圈出錯誤
4. 影片結束後2分鐘小組討論
5. 培訓師揭曉答案+講解

**模擬影片錯誤示範（預錄腳本）：**
```
「同學們，今天我們來學習三角形的周長計算。」❌（應用「周界」）
「這個圖形的寬度是5厘米...」❌（應用「闊度」）
「最後答案就是這樣了。」❌（冇答句）
「大家看這個條形圖...」❌（應用「棒形圖」）
「面積公式就是底乘以高除以二...」❌（冇寫公式步驟）
```
