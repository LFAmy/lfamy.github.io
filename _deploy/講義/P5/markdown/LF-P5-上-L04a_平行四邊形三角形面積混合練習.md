===== WHY BOX：兩條公式擺喺面前，點樣一秒揀啱？ =====

  學完平行四邊形（底×高）同三角形（底×高÷2），
  最大嘅陷阱唔係「唔識計」，而係「用錯公式」！

  呈分試嘅陰濕設計：
      畀一個平行四邊形你，旁邊標數字，然後問面積。
      但佢會同時畀「一條邊」同「一條高」，
      等你用錯數字、或者用錯三角形公式（多除咗個2）！

  一秒公式判斷法（見到圖形即刻做呢三步）：
      ① 望吓個圖形：四邊形（兩對平行邊）→ 平行四邊形 → 底×高
      ② 望吓個圖形：三角形（三個邊三個角）→ 三角形 → 底×高÷2
      ③ 搵底同高：高一定要係「垂直」（90°）嗰條！唔係斜邊！

  最常見嘅自殺式錯誤：
      ❌ 三角形用咗底×高（唔記得÷2） → 答案大咗一倍！
      ❌ 平行四邊形用咗底×高÷2（多除咗2） → 答案細咗一半！
      ❌ 用咗斜邊當高 → 完全錯晒！
      ❌ 已知面積求高，用錯公式方向（應該面積÷底=高）

  記住呢個口訣，終身受用：
      「四邊 = 底乘高，三邊 = 底乘高除二！」
      「見到斜邊唔好用，垂直先係真高度！」

  本堂目標：將呢兩條公式練到變成反射動作，
  唔使諗、唔會錯、呈分試見到面積題直接秒殺！

二、核心知識精講 — 公式選擇三步法 🔴 SSPA

知識點一：圖形識別 → 公式配對（0.5 秒判斷！）

  平行四邊形嘅特徵（見到即刻諗「底×高」）：
      ✅ 四邊形
      ✅ 兩對對邊平行（上下平行、左右平行）
      ✅ 對邊長度相等
      ✅ 對角相等
      → 公式：面積 = 底 × 高

  三角形嘅特徵（見到即刻諗「底×高÷2」）：
      ✅ 三邊形
      ✅ 三個角
      ✅ 任何一邊都可以做底
      → 公式：面積 = 底 × 高 ÷ 2


知識點二：底高對應 — 呈分試最鍾意陰你！🔴🔴🔴

  ⚠️ 一條底只對應一條高！唔可以亂配！

  平行四邊形：
      如果揀咗下面嗰條做底（10cm），
      就要用由呢條底垂直向上度到對邊嘅高度（6cm），
      唔可以用側邊嘅邊長（7cm）！

      底=10cm, 高=6cm → 面積=10×6=60cm² ✅
      底=10cm, 「高」=7cm(斜邊!) → 面積=10×7=70cm² ❌ 錯！


  三角形：
      如果揀咗底邊（12cm），
      就要用由呢條底垂直向上度到對角嘅高度（5cm）。

      底=12cm, 高=5cm → 面積=12×5÷2=30cm² ✅
      底=12cm, 「高」=8cm(斜邊!) → 面積=12×8÷2=48cm² ❌ 錯！


知識點三：已知面積反求 🔄（進階應用）

  如果題目畀咗面積同底，要你求高：

      平行四邊形：高 = 面積 ÷ 底
      三角形：高 = 面積 × 2 ÷ 底  ← 注意！要先×2！

  點解三角形要先×2？
      因為三角形面積 = 底×高÷2
      所以要還原返完整嘅平行四邊形面積先！
      面積×2 = 底×高 → 高 = 面積×2÷底

  例題：三角形面積=24cm²，底=6cm，求高。
      高 = 24×2÷6 = 48÷6 = 8cm ✅
      （常見錯誤：24÷6=4cm ❌ 忘記×2！）

四、SSPA 殺手題實戰（8 分鐘）🔴

SSPA-K1（公式混淆陷阱）：
  題目：計吓呢個三角形面積。
  圖中標示：底=20cm，一條斜邊=15cm，高=12cm（垂直）。
  常見錯誤：20×15÷2=150cm² ❌
  正確：20×12÷2=120cm² ✅
  關鍵：15cm 係斜邊，唔係高！高一定係垂直嗰條 12cm！

SSPA-K2（已知面積反求陷阱）：
  題目：一個三角形面積=60cm²，底=15cm。求高。
  常見錯誤：60÷15=4cm ❌
  正確：60×2÷15=120÷15=8cm ✅
  關鍵：三角形面積=底×高÷2，所以高=面積×2÷底！
        好多人會漏咗×2！

SSPA-K3（雙圖形混合）：
  題目：下圖中，平行四邊形 ABCD 同三角形 CDE 共用底 CD。
  CD=12cm，平行四邊形高=8cm，三角形高=6cm。
  求陰影部分（平行四邊形 + 三角形）的總面積。
  解：平行四邊形=12×8=96cm²
      三角形=12×6÷2=36cm²
      總面積=96+36=132cm² ✅
