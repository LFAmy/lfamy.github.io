二、Core Knowledge （15 min） ＋ Worked Examples
知識點一：連加 = Multiplication（Multiplication的本質）⭐ 最重要！
① Multiplication係相同數字嘅連加嘅捷徑！
② 2 + 2 + 2 + 2 + 2 = 5 個 2 = 2 x 5 = 10
③ a x b = a 連加 b 次 = b 連加 a 次（交換律）
④ 遊戲規則：必須係「相同數字」重複加，先可以轉Multiplication！
例題 1：連加轉Multiplication
🍬 每包糖有 6 粒，買 4 包共有幾粒？
連加：6 + 6 + 6 + 6 = 24
Multiplication：6 x 4 = 24（6 粒 x 4 包 = 24 粒）
交換律驗證：4 x 6 = 24（一樣！）
🪤 例題 2：Trap！唔同數字唔可以轉Multiplication
3 + 5 + 3 + 5 = ？可以寫成Multiplication嗎？
❌ 小Trap怪獸話
3 x 2 + 5 x 2
雖然計到，但已經唔係簡單嘅一個Multiplication算式
✅ 正確理解
唔可以直接變成一個Multiplication
因為加數唔係全部相同！要用連加計：3+5+3+5=16
⚠️ 小Trap怪獸：只有「相同數字重複加」先可以轉Multiplication！例如 2+3+2+3 唔可以變成一個Multiplication，因為數字唔同！
💡 遊戲記憶法：「相同數字聯盟」先可以用Multiplication超能力！唔同數字嘅要乖乖用連加！

知識點三：Multiplication交換律⭐
① a x b = b x a（交換位置，答案一樣！）
② 例：3 x 5 = 5 x 3 = 15
③ 呢個好有用！如果你只記得 5 x 8 = 40，咁 8 x 5 都一定 = 40！
④ Trap注意：Word Problems中，次序有意義！「每人 5 個，3 個人」= 5 x 3 = 15 個
例題 3
用交換律快計：8 x 7 = ？（如果你記得 7 x 8 = 56）
8 x 7 = 7 x 8 = 56（交換律！）

#Common ErrorCorrect Approach
1非相同數字誤用Multiplication：3+3+5 寫成 3x3只有相同數字重複加才可用Multiplication
2Multiplication表記錯：6x8=42（實際=48）每日背誦Multiplication表 5 min，尤其 6-9 的Multiplication
3忘記交換律：只記得 7x8=56，8x7 就唔識交換律：a x b = b x a，記得一個 = 記得兩個
4連加次數數錯用手指指著每個加數，數清楚有幾多個
5Word Problems單位混亂「每人 x 個」係Multiplication，「共」亦係提示
60 的Multiplication忘記：5 x 0 = 0任何數 x 0 = 0，任何數 x 1 = 自己
75 的Multiplication同 10 的Multiplication搞亂5 的Multiple：個位必為 0 或 5
🧠 Mnemonic一：「相同數字連加，快啲轉用Multiplication；幾個幾多乘埋佢，答案即刻出晒黎！」
🧠 Mnemonic二：「二五得十最容易，九嘅Multiplication有祕技；乘數交換答案同，記得一半就夠皮！」
