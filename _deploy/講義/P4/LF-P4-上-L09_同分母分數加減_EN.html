二、Core Knowledge （15 min） ＋ Worked Examples
知識點一：同分母FractionAddition 🔴 SSPA
① 黃金規則：分母相同 → 只加分子，分母不變！
② 例：27 + 37 = 2+37 = 57
③ 帶FractionAddition：整數加整數，Fraction加Fraction。Fraction部分滿咗（分子>=分母）要進位！
④ 絕對不要：連分母都加埋！（27 + 37 ≠ 514 ❌）
🪤 Trap引爆：分母唔可以就咁加！
計算：1/4 + 2/4 = ?
❌ Common Mistake
1/4 + 2/4 = 3/8
（分子加分子·分母加分母）
同分母Fraction相加：只加分子，分母保持不變！
✅ Correct Approach
1/4 + 2/4 = 3/4
分子：1+2=3
分母：4（保持不變）
分母代表「每份有幾大」，相加時份數變多但每份大小不變！
知識點二：同分母FractionSubtraction 🔴 SSPA
① 分母相同 → 只減分子，分母不變！
② 例：58 - 38 = 5-38 = 28 = 14（約簡後）
③ 帶FractionSubtraction：Fraction部分唔夠減 → 向整數借 1（借 1 當做分母咁多份）
  例：314 - 134 = 254 - 134 = 124 = 112
④ 答案要約簡！（P4只要求簡單約簡，如24→12）
例1
310 + 510 = ？（答案 = 810 = 45）
例2
238 + 158 = ？（38+58=88=1 → 3+1=4 → 答案=4）
例3
216 - 56 = ？（16唔夠減56 → 借1 → 176-56=126=113）
⚠️ 世紀災難：27+37=514 ❌！50%學生會連分母都加埋！分母代表每份有幾大，份數增加但每份大小冇變 → 分母不變！
⚠️ SubtractionTrap：58-38唔係等於20（有人減到分母都消失！）。正確= 28=14。

#Common Error（❌ Trap）Correct Approach（✅）
1分母都加埋：25+15=310 ❌分母唔變！只加分子：2+15=35。分母係單位大小！
2答案唔約簡：28 就算要約簡到最簡：28=14。分子分母除公Factor
3帶FractionAddition忘進位：135+245=375 ❌35+45=75=125→3+125=425
4帶FractionSubtraction唔識借位：314-134=?14唔夠減34→借1=借44→254-134=124=112
5整數減Fraction唔識轉：1-35=?1=55→55-35=25
6Subtraction次序搞錯：38-58=28 ❌唔夠減要轉！38-58=負數→呢類題要確認Question係咪合理
7加減混合先後不分加減同一級，由左到右。有括號先計括號。
🧠 Mnemonic：「同分母加減好簡單，分母不變分子計。帶分整數分開做，Fraction滿咗要進位；Fraction唔夠問整數借，借一當做分母份。」
七、4-Step Problem Solving
1
驗分母
分母相同？係→可以直接加減分子。唔係→L10會學點處理。
2
計分子
Addition：分子相加。Subtraction：分子相減。分母保持不變！
3
約簡
分子分母有公Factor→約簡。例：48→12。
4
轉形式
假Fraction→轉帶Fraction。帶Fraction答案→Fraction部分要最簡。
