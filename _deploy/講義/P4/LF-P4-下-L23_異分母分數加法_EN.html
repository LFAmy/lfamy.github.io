二、Core Knowledge （15 min） ＋ Worked Examples
⚠️ 世紀災難Trap！80% 學生第一次都咁錯 ⚠️
1/2 + 1/3 = (1+1)/(2+3) = 2/5？？？❌❌❌ 大錯特錯！
正確答案：1/2 + 1/3 = 3/6 + 2/6 = 5/6 ✅
唔信？半個蛋糕 + 三分一個蛋糕 = 幾乎成個蛋糕（5/6），唔係 2/5（少過一半）！
知識點一：異分母FractionAddition三步法 🔴 SSPA 必考
Step 1 — 通分：搵兩個分母嘅 LCM 做公分母，每個Fraction擴分到公分母
Step 2 — 分子相加：分母相同後，分子相加。分母保持不變！
Step 3 — 約至最簡：如果答案係假Fraction，轉做帶Fraction；如果可以約分，約至最簡
例1
23 + 34 = ？（LCM=12 → 8/12 + 9/12 = 17/12 = 1又5/12）
例2
58 + 16 = ？
❌ Trap1：直接加分子分母
5/8 + 1/6 → (5+1)/(8+6)=6/14
錯！分母唔同，份唔一樣大
✅ 正確：先通分再加分子
LCM=24 → 15/24 + 4/24
= 19/24 ✅
知識點二：帶FractionAddition 🔴 SSPA
① 方法一（推薦）：整數加整數 + Fraction加Fraction。Fraction部分需要通分
  例：112 + 223 = (1+2) + (36 + 46) = 3 + 76 = 3 + 116 = 416
② 方法二：全部轉成假Fraction → 通分 → 相加 → 轉回帶Fraction
例3
134 + 216 = ？（整數部分：1+2=3；Fraction部分：LCM=12 → 9/12+2/12=11/12；答：3又11/12）
知識點三：三個Fraction相加 + 約分驗收 🟡 SSPA
① 三個Fraction：搵三個分母嘅 LCM → 全部擴分 → 分子相加 → 約至最簡
② 約分驗收標準：答案一定要約到最簡！如果分子分母有公Factor → 再約！
③ 假Fraction答案 → 必須轉做帶Fraction（呈分試要求）
例4
12 + 13 + 14 = ？LCM(2,3,4)=12

每個漢堡包代表一個Fraction。一層層疊起（通分），總高度 = 和！睇邊個最快砌出正確漢堡。

#Common Error（❌ Trap）Correct Approach（✅）
1世紀災難：分子+分子·分母+分母先通分！分母相同先可以分子相加，分母保持不變
2忘記通分：異分母直接加分子搵LCM→擴分→分母相同→分子相加→分母不變
3通分後分母相加：3/6+2/6=5/12（❌）3/6+2/6 = (3+2)/6 = 5/6。分母相同所以不變！
4帶Fraction：整數加Fraction但忘記Fraction部分要通分Fraction部分必須通分！整數部分可直接相加
5Fraction部分相加後 >=1 沒有進位Fraction部分若 >=1，轉假Fraction→帶Fraction→整數進位
6答案冇約至最簡檢查分子分母有冇公Factor。6/8 → 3/4
7假Fraction答案冇轉帶Fraction呈分試要求！17/12 → 1又5/12
🧠 Mnemonic：「異分母Addition有秘方，先搵LCM做公分母。通分擴分要同步，分子相加分母留。答案記得約最簡，假分轉帶先收手。」
七、4-Step Problem Solving
1
搵LCM
睇分母→搵LCM→呢個就係公分母。LCM=最細公Multiple
2
通分擴分
每個Fraction：分母乘幾多→分子乘同樣數。分母變LCM
3
分子相加
分母相同→只加分子。分母保持不變！（呢步最易錯）
4
約簡轉帶
約至最簡→假Fraction轉帶Fraction→完成！驗算答案合理性
