二、Core Knowledge （15 min） ＋ Worked Examples
知識點一：Fraction的三種類型 🔴 SSPA
① 真Fraction：分子 < 分母。數值 < 1。例：23、58
② 假Fraction：分子 >= 分母。數值 >= 1。例：53、88
③ 帶Fraction：整數 + 真Fraction。例：123（讀作「一又三分之二」）
④ 假Fraction和帶Fraction代表同一個數值！可以互換！
🪤 Trap引爆：分母唔可以就咁加！
計算：1/4 + 2/4 = ?
❌ Common Mistake
1/4 + 2/4 = 3/8
（分子加分子·分母加分母）
同分母Fraction相加：只加分子，分母保持不變！
✅ Correct Approach
1/4 + 2/4 = 3/4
分子：1+2=3
分母：4（保持不變）
分母代表「每份有幾大」，相加時份數變多但每份大小不變！
知識點二：假Fraction ⇄ 帶Fraction互換 🔴 SSPA
① 假Fraction → 帶Fraction：分子 ÷ 分母 → 商 = 整數部分，餘數 = 新分子，分母不變
  例：73 = 7÷3 = 2餘1 → 213
② 帶Fraction → 假Fraction：整數 × 分母 + 分子 → 寫在分子，分母不變
  例：213 = (2×3+1)/3 = 7/3
③ 假Fraction = 帶Fraction，只係寫法唔同！兩者數值完全一樣
例1
把 114 轉換成帶Fraction。（11÷4=2餘3 → 234）
例2
把 325 轉換成假Fraction。（3×5+2=17 → 175）
⚠️ 致命Trap：分子分母方向！23 ≠ 32！分子在上、分母在下。調轉哂 = 完全唔同的數！
⚠️ 55 係真Fraction定假Fraction？分子=分母 → 數值=1 → 假Fraction！因為假Fraction定義係分子>=分母。

3假Fraction轉帶Fraction計錯商：11÷4=2餘3→寫2311 ❌分母唔會變！11÷4=2餘3→234 ✓

🧠 Mnemonic：「分子在上分母下，真細假大帶分夾。假轉帶除睇商餘，帶轉假乘加分子。」
七、4-Step Problem Solving
1
辨類型
分子<分母→真。分子>=分母→假。有整數+Fraction→帶。
2
歸一類
比較或計算時，全部轉做同一類型。通常轉做假Fraction最方便。
3
互換計
假→帶：分子÷分母=商餘數。帶→假：整數×分母+分子。
4
驗方向
真Fraction < 1，假Fraction >= 1。計完檢查：你個答案合理嗎？
