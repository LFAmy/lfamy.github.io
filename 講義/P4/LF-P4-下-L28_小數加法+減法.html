二、核心知識精講 （15 分鐘） ＋ 例題練習
⚠️ 致命陷阱：小數點對齊 vs 右對齊
整數加減用「右對齊」（個位對個位），但小數加減要用「小數點對齊」！
❌ 錯：0.5 同 0.25 右對齊 → 5對5、0對2 → 0.5+0.25=0.30（錯！）
✅ 對：0.50 + 0.25 → 小數點對齊 → 0.75（正確！）
知識點一：小數加法 🔴 SSPA 必考
① Step 1 — 小數點對齊：兩個數的小數點要喺同一條垂直線上
② Step 2 — 補零：小數位唔夠→補0。例：0.5 + 0.25 → 0.50 + 0.25
③ Step 3 — 如常加法：由右至左逐位加，跟整數加法一樣
④ Step 4 — 小數點落返：答案的小數點 = 同對齊位置落返
WHY BOX：小數除法的本質 — 把小數變成整數來除
核心技巧：6.09 / 3 = ?
609 / 3 = 203 → 小數點向左移 2 位 → 2.03 ✅
常見陷阱：6.09 / 3 ≠ 2.3！小數點位置最容易錯。
驗算：答案 × 除數 = 被除數 → 2.03 × 3 = 6.09 ✓
💡 小數加減黃金法則：小數點對齊！對齊小數點後，當成整數加減，最後在答案的相同位置點上小數點。位數不同可以補0。
💡 小數加減黃金法則：小數點對齊！對齊小數點後，當成整數加減，最後在答案的相同位置點上小數點。位數不同可以補0。

知識點二：小數減法 🔴 SSPA 必考
① 同加法一樣：小數點對齊 + 補零
② 借位同整數減法一樣：唔夠減就向前借1。借1 = 借嗰位嘅10個單位
③ 例：0.82 - 0.56 → 百分位：2唔夠減6 → 借位：12-6=6 → 十分位：7-5=2 → 0.26
④ 整數減小數：整數後面加 .0（或 .00）再減
例3
0.73 - 0.48 = ？（借位：百分位 3-8 唔夠→向十分位借→13-8=5；十分位 6-4=2 → 0.25）
例4
2 - 0.65 = ？（2 = 2.00 → 2.00 - 0.65 = 1.35）
知識點三：小數加減混合 + 驗算 🟡 SSPA
① 由左至右：同整數一樣，冇括號就由左至右逐個計
② 驗算方法：加法用減法驗（和 - 加數 = 另一加數）。減法用加法驗（差 + 減數 = 被減數）
③ 估算先驗：計之前先估算→計完睇答案係咪合理
例5
3.5 + 2.78 - 4.1 = ？（補零→ 3.50+2.78=6.28 → 6.28-4.10=2.18）

1右對齊（同整數）：0.5+0.25→ 5對5 錯位小數點對齊！0.50+0.25。小數點在垂直線上
2冇補零：0.5+0.25 計成 0.5+0.25=0.30（❌）補0：0.50+0.25。十分位5+2=7→0.75
3小數點忘記落返：答案 075 冇點答案的小數點要同對齊位置垂直落返
4借位時忘記減1：0.82-0.56，借位後十分位仍是8借位=借嗰位減1。8→7（借咗1俾百分位）
5整數減小數：2-0.65=1.65?（❌）2=2.00→2.00-0.65=1.35

7答案末位零冇刪：3.50 寫成 3.5（可以）但最好不要P4可保留末位零以清晰表示準確度。3.5=3.50
🧠 口訣：「小數加減唔同整，小數點對齊先開始。位數唔同補個零，由右至左逐位計。借位同整數一樣做，小數點直落唔會錯。」
七、解題四步卡
1
對齊小數點
兩個數小數點喺同一垂直線。這是唯一正確的寫法！
2
補零
小數位數不同→在短的數後面補0。0.5 + 0.25 → 0.50 + 0.25
3
逐位加減
由右至左（百分位→十分位→個位...）逐位計算。借位照做
4
落小數點
答案的小數點 = 喺對齊位置直落。驗算：估算覆核
