二、Core Knowledge （15 min） ＋ Worked Examples
知識點一：Decimal點後的世界 — 十分位 🔴 SSPA
① Decimal點 "." 係整數同Decimal嘅分界線
② Decimal點後第一位 = 十分位（1/10）
  例：0.3 = 3 個 1/10 = 3/10。讀作「零點三」
③ 十分位嘅值：1 個十分位 = 0.1 = 1/10。10 個 0.1 = 1.0
④ 0.5 = 一半。0.5 = 5/10 = 1/2（約簡後）
知識點二：百分位 — Decimal點後第二位 🔴 SSPA
① Decimal點後第二位 = 百分位（1/100）
  例：0.03 = 3 個 1/100 = 3/100
② 百分位嘅值：1 個百分位 = 0.01 = 1/100。100 個 0.01 = 1.0
③ 十份位和百分位嘅關係：0.1 = 0.10（十分位=10個百分位）
④ 0.25 = 25/100 = 1/4（約簡後）
例1
在 0.47 中，4 在哪個位？7 在哪個位？各代表多少？
例2
寫出Decimal：3 個 0.1 + 5 個 0.01 = ？
❌ Trap：0.1 vs 0.01 混淆
以為 0.1 = 1/100（❌）
0.5 > 0.05？分唔清！
✅ 正確
0.1 = 1/10（Decimal點後第1位=十分位）
0.01 = 1/100（第2位=百分位）
知識點三：Fraction ↔ Decimal互換 🔴 SSPA 必考
① 分母10的Fraction → Decimal：分子放十分位。3/10 = 0.3
② 分母100的Fraction → Decimal：分子放十+百分位。25/100 = 0.25
③ Decimal → Fraction：0.7 = 7/10（可約簡）。0.45 = 45/100 = 9/20（約簡）
④ 常用記住：0.5=1/2, 0.25=1/4, 0.75=3/4, 0.2=1/5, 0.1=1/10
例3

知識點四：Decimal位值表 — 擴展整數位值概念 🟡 SSPA
① Decimal點左邊 = 整數部分（個、十、百...）。右邊 = Decimal部分（十分、百分...）
② Decimal點後可以有多位：0.1（1位）、0.01（2位）、0.001（3位=千分位）
③ P4 重點：1-2 位Decimal。P5 會學 3 位。

#Common Error（❌ Trap）Correct Approach（✅）
1十分位同百分位混淆：0.25 以為 2=百分位Decimal點後第1位=十分位(1/10)，第2位=百分位(1/100)
20.1 同 0.01 搞錯：0.1 = 1/100（❌）0.1=1/10（一位Decimal）。0.01=1/100（兩位Decimal）
3讀Decimal漏讀「點」：0.37 讀「零三七」正確讀法：「零點三七」
4Fraction轉Decimal分母搞錯：3/100=0.3（❌）3/100=0.03。分母100→兩個Decimal位
5Decimal轉Fraction冇約簡：0.5=5/10（可以約）5/10=1/2。一定要約到最簡！
6Decimal點位置錯：寫成 .5 或 0.5.（多咗點）標準寫法：0.5。前面要有0，後面唔要點
7$12.5 讀錯：十二元五十二元五毫。$12.50=$12元5毫（Decimal後兩位代表毫）
🧠 Mnemonic：「Decimal點後第一位係十分，第二位係百分。零點一係十分一，零點零一係百分一。零點五係一半，零點二五係四分一。」
七、4-Step Problem Solving
1
睇Decimal點
Decimal點前=整數，後=Decimal。第1位=1/10，第2位=1/100
2
分→小
分母10→1位Decimal。分母100→2位Decimal。分子寫係對應位置
3
小→分
睇有幾多位→分母係10/100。分子=Decimal數字。最後要約簡
4
驗算
讀一次。檢查Decimal點位置。FractionDecimal互換核對數值
