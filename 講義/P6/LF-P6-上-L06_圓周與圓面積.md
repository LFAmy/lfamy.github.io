===== WHY BOX：π（圓周率）——數學界最神秘嘅數字 =====

  π = 3.1415926535... 永遠除唔盡、冇規律、冇盡頭！

  任何圓形，圓周÷直徑 = π（永遠不變！）
  無論個圓有幾大：一個汽水蓋定係地球，呢個比例都係 π！

  小學用兩個近似值：
      π ≈ 3.14（一般計算用）
      π ≈ 22/7（當半徑/直徑係7的倍數時更方便）

  兩條公式：
      圓周 = π × 直徑   →  C = πd
      圓周 = 2 × π × 半徑 → C = 2πr
      
  ⚠️ 圓周單位係 cm/m（長度單位），唔係 cm²/m²！
     面積先有平方，圓周冇平方！

  口訣：「圓周=拍直徑，半徑要乘2！」

===== 二、核心知識精講 ＋ 例題練習 =====

--- 知識點一：圓的複習（圓心、半徑、直徑） 🔴 SSPA ---

![圓的各部分](../_svg/circle_parts_radius_diameter.svg)

| 名稱 | 符號 | 定義 | 關係 |
|------|:--:|------|------|
| 圓心 | O | 圓的正中心點 | — |
| 半徑 | r | 圓心到圓周的距離 | r = d ÷ 2 |
| 直徑 | d | 穿過圓心、兩端在圓周的線段 | d = 2r |
| 圓周 | C | 圓的周界長度 | C = πd = 2πr |

> 🔑 關鍵：圓規兩腳的距離 = 半徑！唔係直徑！

例題1：一個圓，半徑 = 7cm。求：直徑 = ？
解：d = 2 × r = 2 × 7 = 14cm  ← 就係咁簡單！

例題2：一個圓，直徑 = 18cm。求：半徑 = ？
解：r = d ÷ 2 = 18 ÷ 2 = 9cm

--- 知識點二：圓周率 π —— 數學最神奇嘅常數 🔴 SSPA ---

π（讀音：拍 / pai）係一個永遠唔會完嘅數字：
  π = 3.14159265358979323846...

最重要嘅特性：
  🔑 任何圓形，圓周 ÷ 直徑 = π（呢個比例永遠不變！）

小學用兩個 π 近似值：
| 近似值 | 用咩時候 | 例子 |
|:--:|------|------|
| π ≈ 3.14 | 一般計算（半徑/直徑唔係7的倍數） | r=5cm → 用3.14 |
| π ≈ 22/7 | 半徑或直徑係7的倍數 | r=7cm, d=14cm → 用22/7 |

> ⚠️ 點解用22/7？因為22/7 ≈ 3.142857... ≈ 3.14，而且可以同7約分，計起嚟快好多！

例題3：圓周÷直徑 = 3.14159...，呢個數叫咩？
答：π（圓周率）

--- 知識點三：圓周公式 🔴 SSPA 必考！---

兩條公式，記住一條就得（因為 d = 2r）：

| 公式 | 用咩情況 | 例子 |
|------|------|------|
| C = π × d | 題目畀直徑 | d=10cm → C = 3.14×10 = 31.4cm |
| C = 2 × π × r | 題目畀半徑 | r=5cm → C = 2×3.14×5 = 31.4cm |

> 兩條公式答案一樣！因為 d = 2r，所以 πd = π(2r) = 2πr

例題4（SSPA題型）：一個圓形花圃，半徑 = 7m。求圓周。（取 π = 22/7）
解：C = 2πr = 2 × (22/7) × 7
       = 2 × 22 × (7÷7)
       = 2 × 22 × 1
       = 44m  ← 22/7 的好處：7可以約分！

例題5（SSPA題型）：一個圓形時鐘，直徑 = 30cm。求圓周。（取 π = 3.14）
解：C = πd = 3.14 × 30
       = 94.2cm

--- 知識點四：已知圓周，反求直徑/半徑 🔴 SSPA 殺手級！---

| 求甚麼 | 公式 | 口訣 |
|------|------|------|
| 已知C求d | d = C ÷ π | 圓周÷π = 直徑 |
| 已知C求r | r = C ÷ π ÷ 2 | 圓周÷π÷2 = 半徑 |

例題6：一條鐵線圍成一個圓形，圓周 = 62.8cm。求半徑。（取 π = 3.14）
解：Step 1: d = C ÷ π = 62.8 ÷ 3.14 = 20cm
    Step 2: r = d ÷ 2 = 20 ÷ 2 = 10cm
    
    或一步過：r = 62.8 ÷ 3.14 ÷ 2 = 10cm

例題7（陷阱！）：圓周 = 44cm，求直徑。（取 π = 22/7）
解：d = C ÷ π = 44 ÷ (22/7)
       = 44 × (7/22)
       = (44×7) ÷ 22
       = 308 ÷ 22
       = 14cm  ← 用22/7時，除以分數 = 乘以倒數！

一個圓形水池，直徑 = 14m。小明沿水池邊緣走了 2.5 圈，共走了多少米？（取 π = 22/7）

===== 五、課堂總結 + 必記口訣（3 分鐘）=====

📌 圓心O · 半徑r · 直徑d=2r · 圓周C
📌 π ≈ 3.14（一般）或 22/7（有7的倍數時）
📌 圓周公式：C = πd = 2πr
📌 反求：d = C÷π，r = C÷π÷2
📌 ⚠️ 圓周單位係 cm/m（長度）！冇平方！
📌 口訣：「圓周 = 拍直徑，半徑前面乘個2」
📌 補習名師溫馨提示：π=22/7 只喺半徑/直徑係7的倍數時先用，其他情況一律用3.14！
