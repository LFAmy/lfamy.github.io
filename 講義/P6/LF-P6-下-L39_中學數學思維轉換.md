===== WHY BOX：小學數學 vs 中學數學——完全唔同嘅遊戲規則 =====

  小學數學：計出答案 = 完成（答案啱就得）
  中學數學：過程 = 分數嘅 70%！（答案啱但過程錯 = 零分）

  中學數學三大轉變：

  ① 從「計數」到「證明」
      小學：3×5=15（寫答案）
      中學：證明「三角形內角和=180°」
            → 畫輔助線→寫理由→逐步推導→結論
            每一行都要寫「原因」！

  ② 從「數字」到「符號」
      小學：x + 3 = 7 → x = 4
      中學：3x² + 2x - 5 = 0 → 用公式/因式分解
            ax² + bx + c = 0（a,b,c 係任何數嘅通用解）

  ③ 從「一題一解」到「一題多解」
      小學：一條題目通常有一種解法
      中學：同一條題目可以用代數/幾何/坐標多種方法
            → 揀最簡潔嗰種先係高手！


一、中學答題格式訓練

例題：證明「對頂角相等」

小學寫法（唔合格）❌：
  「角A = 角B，因為睇落一樣」

中學寫法（滿分）✅：
  已知：AB與CD為相交直線
  求證：∠1 = ∠3
  證明：
      ∠1 + ∠2 = 180°（直線上的鄰角）
      ∠3 + ∠2 = 180°（直線上的鄰角）
      ∴ ∠1 = ∠3（等量公理）
  Q.E.D.


二、中學常見題型預覽

1. 因式分解：x² - 9 = (x+3)(x-3)
2. 聯立方程：{x + y = 7, x - y = 3} → x=5, y=2
3. 坐標幾何：求(2,3)同(5,7)之間的距離
4. 百分法進階：複利息計算
5. 統計：平均值、中位數、眾數、分佈


三、暑期中學預習建議

必讀（7-8月，每日30分鐘）：
  □ 負數四則運算 → 熟到變反射
  □ 代數式化簡 → 合併同類項、去括號
  □ 一元一次方程 → 搬項技巧
  □ 坐標系 → 四個象限、距離公式

選讀（學有餘力）：
  □ 因式分解入門
  □ 聯立方程入門
  □ 不等式進階
