===== WHY BOX：列方程最難嘅一步——「設 x 應該設邊個？」 =====

  呢個係由「計數」升級到「數學建模」嘅關鍵判斷力！

  黃金法則：通常設「最細嗰個」或者「問題問嗰個」做 x！

  例子 1（年齡問題）：
      「爸爸年齡係小明嘅 3 倍多 2，兩人加埋 54 歲。小明幾歲？」
      設小明 = x（最細嗰個）→ 爸爸 = 3x+2
      x + (3x+2) = 54 → 4x = 52 → x = 13 ✅

  例子 2（分錢問題）：
      「甲、乙、丙三人分 $1200。甲係乙嘅 2 倍，乙係丙嘅 3 倍。」
      設丙 = x（最細嗰個！）→ 乙 = 3x → 甲 = 6x
      6x + 3x + x = 1200 → 10x = 1200 → x = 120
      甲=$720，乙=$360，丙=$120 ✅

  如果設錯 x（例如設甲=x），條式會變得好複雜！

  口訣：「有倍數關係→設最細嗰個做 x，其餘用 x 表達！」

二、核心知識精講 ＋ 例題練習

知識點一：年齡問題 🔴 SSPA

  ① 設而家年齡（通常設最細嗰個）
  ② 「n 年後」→每人加 n
  ③ 「n 年前」→每人減 n
  ④ 年齡差永遠不變！

  例題 1：爸爸今年 40 歲，小明今年 12 歲。
           幾年後爸爸年齡係小明嘅 3 倍？
      設 x 年後：40+x = 3(12+x)
      40+x = 36+3x → 4 = 2x → x = 2
      2年後爸爸42歲、小明14歲，42=3×14 ✅


知識點二：分配/倍數問題 🔴 SSPA

  ① 搵「最細單位」設 x
  ② 按倍數關係寫出其他量
  ③ 總和=已知總數→列方程

  例題 2：蘋果、橙、香蕉共 180 個。蘋果係橙嘅 2 倍，
           橙係香蕉嘅 3 倍。各幾多個？
      設香蕉 = x → 橙 = 3x → 蘋果 = 6x
      x + 3x + 6x = 180 → 10x = 180 → x = 18
      香蕉 18、橙 54、蘋果 108 ✅


知識點三：幾何+方程綜合 🔴 SSPA

  ① 用代數表示邊長/角度/面積
  ② 利用幾何關係列方程（周界/面積/角度和）
  
  例題 3：長方形長比闊多 5cm，面積=84cm²。求長和闊。
      設闊 = x → 長 = x+5
      x(x+5) = 84 → x²+5x-84=0 → (x+12)(x-7)=0
      x = 7（x 不能為負）
      闊=7cm，長=12cm ✅

四、SSPA 殺手題實戰 🔴

SSPA-K1（經典：雞兔同籠方程版）：
  農場有雞同兔共 35 隻，腳共 94 隻。雞兔各幾多？
  設雞 = x → 兔 = 35-x
  2x + 4(35-x) = 94 → 2x+140-4x=94 → -2x=-46 → x=23
  雞 23 隻，兔 12 隻 ✅

SSPA-K2（百分數+方程）：
  一件貨品減價 25% 後賣 $180。原價係？
  設原價 = x → x × 0.75 = 180 → x = 240 ✅
