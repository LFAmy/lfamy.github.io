===== WHY BOX：負數——數學世界嘅「反物質」 =====

  小學六年嚟，你學嘅數字全部都係「正數」：1, 2, 3, 1/2, 0.5...
  但真實世界有好多「相反方向」嘅概念！

  負數嘅真實意義：
      🌡️ 溫度：0°C 以下 → -5°C（比 0 仲要凍 5 度）
      💰 金錢：欠錢 → -$100（負債 $100）
      🏢 樓層：地庫 → -1 樓（地面以下）
      📉 股票：跌咗 → -10%（蝕咗 10%）

  數線嘅概念：
      ←───|───|───|───|───|───|───→
        -3  -2  -1   0   1   2   3
      
      向右行 = 加（數值變大）
      向左行 = 減（數值變小）
      跨越 0 = 正負號轉變！

  最重要嘅概念：負負得正！
      -(-5) = +5（「負」嘅相反 = 正）
      3 - (-2) = 3+2 = 5（減負數 = 加正數）

  口訣：「數線右加左減，負負得正，減負等於加正！」

二、核心知識精講

知識點一：負數加減（數線法）🟢

  ① 加法：向右行
      -3 + 5：由 -3 向右行 5 格 → 2 ✅
      5 + (-3)：由 5 向左行 3 格 → 2 ✅
  
  ② 減法：向左行
      3 - 5：由 3 向左行 5 格 → -2 ✅
      -2 - 3：由 -2 向左行 3 格 → -5 ✅
  
  ③ 減負數 = 加正數
      4 - (-3) = 4 + 3 = 7 ✅（兩個負號變加號！）
      -5 - (-2) = -5 + 2 = -3 ✅


知識點二：代數式化簡 🟢

  ① 同類項：相同字母同次數嘅項可以合併
  ② 合併 = 係數相加，字母不變
  
  例：3x + 5x = 8x ✅
      2x² + 3x² = 5x² ✅
      3x + 2y + 5x = 8x + 2y ✅（x同y唔同類，唔可以合併！）
      4x - x = 3x ✅（x前面隱藏係數1）


知識點三：去括號進階 🟢

  ① 括號前係正號：直接去括號，入面唔變
  ② 括號前係負號：去括號時入面每項正負反轉！
  
  例：3x + (2x - 5) = 3x + 2x - 5 = 5x - 5 ✅
      3x - (2x - 5) = 3x - 2x + 5 = x + 5 ✅（-(-5)=+5！）
