===== WHY BOX：不等式——唔係「等於」，而係「大過/細過」！ =====

  以前你解方程：x + 3 = 7 → x = 4（只有一個答案）
  而家不等式：x + 3 > 7 → x > 4（無限個答案！）

  不等式係現實世界最常用嘅數學：
      🎂 生日：「必須 12 歲或以上」→ 年齡 ≥ 12
      🎢 過山車：「身高最少 140cm」→ 身高 ≥ 140
      🅿️ 停車場：「限高 2m」→ 車高 ≤ 2
      🛒 消費：「買夠 $200 免運費」→ 消費 ≥ 200

  最重要嘅規則（中一必考！）：
      不等式兩邊乘或除「負數」→ 不等號要反方向！
      例如：-2x > 6 → x < -3（除咗 -2，> 變 <！）
      點解？因為乘/除負數會將數線「反轉」！

  口訣：「不等式乘除負數，大過變細過，細過變大過！」

二、核心知識精講

知識點一：不等式符號與解不等式 🟢

  符號：
      >  大過（greater than）
      <  細過（less than）
      ≥  大過或等於（at least / 不少於）
      ≤  細過或等於（at most / 不超過）
  
  解不等式＝解方程，但有一個關鍵例外！

  例：x + 5 ≥ 12 → x ≥ 7 ✅
      3x < 15 → x < 5 ✅
      但！-2x > 8 → x < -4 ✅（除負數→反方向！）


知識點二：為什麼乘除負數要反方向？🟢🔴

  用實際數字驗證：
      不等式：2 < 5（真確）
      兩邊乘 -1：-2 ？ -5
      -2 同 -5 邊個大？-2 > -5（因為 -2 喺數線右邊！）
      所以：原本 < 變成 >！
      呢個就係「乘除負數要反方向」嘅原因！


知識點三：坐標系四象限 🟢

            y軸
            ↑
    第二象限 │ 第一象限
    (-, +)   │  (+, +)
    ─────────┼─────────→ x軸
    第三象限 │ 第四象限
    (-, -)   │  (+, -)

  記法：象限由右上開始，逆時針數（1→2→3→4）
