===== WHY BOX：扇形 = 圓形嘅「一塊pizza」？環形 =「冬甩」？=====

  扇形：想像一個圓形pizza，切出一塊——
     呢塊pizza嘅面積 = 成個pizza面積 × (呢塊嘅角度÷360°)
     
     例如：90°扇形 = ¼ 個圓（因為90/360 = 1/4）
          60°扇形 = ⅙ 個圓（因為60/360 = 1/6）
  
  環形：想像一個冬甩（donut）——
     外圈係大圓，內圈係小圓（個窿）
     環形面積 = 大圓面積 − 小圓面積
     
     關鍵：兩個圓嘅圓心係同一個點！
  
  複合圓形：將扇形、半圓、¼圓、環形拼埋一齊
     → 拆！逐part計！先拆後加減！
  
  口訣：「扇形=圓×分數，環形=大減小，複合=拆拆拆！」

===== 二、核心知識精講 ＋ 例題練習 =====

--- 知識點一：扇形面積 🔴 SSPA ---

扇形面積 = 圓面積 × (圓心角 ÷ 360°)

| 圓心角 | 佔圓的幾分之幾 | 扇形面積公式 |
|:--:|:--:|------|
| 90° | 90/360 = 1/4 | A = ¼ × πr² |
| 60° | 60/360 = 1/6 | A = ⅙ × πr² |
| 45° | 45/360 = 1/8 | A = ⅛ × πr² |
| 120° | 120/360 = 1/3 | A = ⅓ × πr² |
| 180° | 180/360 = 1/2（半圓！） | A = ½ × πr² |

> 🔑 先約分！360度可以同好多數字約分！

例題1：一個扇形，半徑=10cm，圓心角=90°。求面積。（取 π = 3.14）
解：Step 1: 求整個圓面積 = 3.14 × 10² = 314cm²
    Step 2: 扇形 = 314 × (90/360) = 314 × ¼ = 78.5cm²

例題2：一個扇形，半徑=7cm，圓心角=60°。求面積。（取 π = 22/7）
解：整個圓面積 = (22/7) × 7² = 154cm²
    扇形 = 154 × (60/360) = 154 × ⅙
    
    妙計：154 ÷ 6 = 25.666... 
    用分數：154/6 = 77/3 ≈ 25.67cm²
    
    或用 3.14: A = 3.14 × 49 × (60/360)
                  = 153.86 × ⅙ ≈ 25.64cm²
    
    ⚠️ 用22/7時如果除唔盡，建議轉用3.14或留分數！

--- 知識點二：扇形周界 🔴 SSPA ---

⚠️ 扇形周界 = 弧長 + 2條半徑（唔係扇形面積！）

弧長 = 圓周 × (圓心角 ÷ 360°)

| 要求 | 公式 |
|------|------|
| 扇形面積 | A = πr² × (θ/360°) |
| 扇形弧長 | L = 2πr × (θ/360°) |
| 扇形周界 | P = 弧長 + 2r |

例題3：扇形 r=10cm，圓心角=90°，求周界。（取 π = 3.14）
解：弧長 = 2×3.14×10 × (90/360) = 62.8 × ¼ = 15.7cm
    周界 = 15.7 + 2×10 = 15.7 + 20 = 35.7cm

例題4：半圓，直徑=14cm，求面積和周界。（取 π = 22/7）
解：r = 7cm
    面積 = ½ × (22/7) × 7² = ½ × 154 = 77cm²
    弧長 = ½ × 2×(22/7)×7 = ½ × 44 = 22cm
    周界 = 22 + 14 = 36cm  ← 半圓周界 = 弧長 + 直徑！

--- 知識點三：環形面積 🔴 SSPA ---

環形面積 = 大圓面積 − 小圓面積

> ⚠️ 環形最常見陷阱：題目畀直徑，但計面積要用半徑！
> 外圓直徑 D → 外圓半徑 R = D÷2
> 內圓直徑 d → 內圓半徑 r = d÷2
> 環形 = πR² − πr² = π(R²−r²)

例題5：一個環形，外圓半徑=8cm，內圓半徑=5cm。求環形面積。（取 π = 3.14）
解：外圓面積 = 3.14 × 8² = 3.14 × 64 = 200.96cm²
    內圓面積 = 3.14 × 5² = 3.14 × 25 = 78.5cm²
    環形面積 = 200.96 − 78.5 = 122.46cm²

例題6（陷阱！）：環形，外圓直徑=20cm，內圓直徑=12cm。求環形面積。（取 π = 3.14）
⚠️ 陷阱：20同12係直徑！要先÷2！
解：R = 20÷2 = 10cm，r = 12÷2 = 6cm
    環形 = 3.14×(10²−6²) = 3.14×(100−36) = 3.14×64 = 200.96cm²

--- 知識點四：複合圓形 🔴 SSPA 壓軸級！---

策略：拆！將複合圖形拆成基本圖形（圓/半圓/扇形/正方形/長方形/三角形）

例題7：一個正方形邊長=14cm，四個角各有一個¼圓（半徑=7cm）。求陰影部分面積。（取 π = 22/7）
解：正方形面積 = 14×14 = 196cm²
    4個¼圓 = 1個完整圓！面積 = (22/7)×7² = 154cm²
    陰影 = 196 − 154 = 42cm²

例題8：一個長方形（20cm×10cm），兩端各有一個半圓（直徑=10cm）。
求整個圖形的面積和周界。（取 π = 3.14）
解：長方形部分：如果半圓在左右兩端，
    長方形 = 20×10 = 200cm²（但注意：半圓佔據了兩端的部分）
    
    兩個半圓 = 1個完整圓！r=5cm
    圓面積 = 3.14×5² = 78.5cm²
    總面積 = 200 + 78.5 = 278.5cm²
    
    周界：圓周(兩個半圓=一個圓周) + 兩條長邊
    = 2×3.14×5 + 2×20 = 31.4 + 40 = 71.4cm

SSPA-K2（呈分試2021改編）：

r² = 113.04 ÷ 3.14 = 36
