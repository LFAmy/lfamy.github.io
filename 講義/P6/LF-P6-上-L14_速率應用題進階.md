===== WHY BOX：點解平均速率唔係「速率嘅平均」？=====

  小明由A去B：速率60km/h（用咗1小時）
  小明由B返A：速率40km/h（用咗1.5小時）
  
  錯誤計法：(60+40)÷2 = 50km/h ← 錯！
  
  正確計法：總距離 ÷ 總時間
      距離AB = 60×1 = 60km（來回一樣）
      總距離 = 120km，總時間 = 1+1.5 = 2.5h
      平均速率 = 120÷2.5 = 48km/h ← 正確！
  
  點解唔同？因為兩段所用嘅時間唔同！
  平均速率永遠 = 總距離÷總時間
  
  口訣：「平均速率=總距÷總時，唔係速率嘅平均！」

===== 二、核心知識精講 =====

--- 知識點一：平均速率（唔係速率平均！）🔴 ---

平均速率 = 總距離 ÷ 總時間

例題1：車由A去B：速率80km/h用2h；由B返A：速率60km/h。求全程平均速率。
解：AB距離 = 80×2 = 160km
    回程時間 = 160÷60 = 8/3 ≈ 2.67h
    總距離 = 320km，總時間 = 2+2.67 = 4.67h
    平均速率 = 320÷4.67 ≈ 68.6km/h

--- 知識點二：多段行程（分段計！）🔴 ---

例題2：全程240km。前1/3用速率60km/h，後2/3用速率80km/h。求全程平均速率。
解：前段距離 = 240×⅓ = 80km，時間 = 80÷60 = 4/3h
    後段距離 = 240×⅔ = 160km，時間 = 160÷80 = 2h
    總時間 = 4/3+2 = 10/3h
    平均速率 = 240÷(10/3) = 240×3/10 = 72km/h

--- 知識點三：速率圖表結合 🔴 ---

例題3：折線圖顯示距離-時間關係。0-2h：行80km；2-5h：休息；5-8h：行120km。
(a) 0-2h速率？(b)全程平均速率？
解：(a) 80÷2 = 40km/h
    (b) 總距=200km，總時=8h，平均=25km/h

--- 知識點四：相遇和追及 🔴 ---

相遇：總距離 = (速率A+速率B) × 時間
追及：距離差 = (速率A−速率B) × 時間

例題4：A、B兩城相距300km。甲車由A出發(60km/h)，乙車同時由B出發(40km/h)相向而行。幾小時後相遇？
解：相遇時間 = 300÷(60+40) = 3小時

===== 五、課堂總結 =====
📌 平均速率 = 總距離÷總時間（非速率平均！）
📌 多段行程：逐段計時間，再加總
📌 相遇：時間=總距÷速率和
📌 追及：時間=距離差÷速率差
