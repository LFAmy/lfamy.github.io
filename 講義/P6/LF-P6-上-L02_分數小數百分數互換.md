===== WHY BOX：點解要學三種寫法互換？ =====

  分數、小數、百分數——其實係同一樣嘢嘅三種「語言」！

  1/2 = 0.5 = 50% → 全部都係「一半」！

  咩時候用邊種？
      分數：精確計算（例如 1/3 = 0.333... 冇盡頭）
      小數：方便比較大小（0.375 vs 0.4 一眼睇到邊個大）
      百分數：表達比例（「75%嘅人同意」比「3/4嘅人同意」更直觀）

  互換黃金法則：
      分數→小數：分子÷分母
      小數→百分數：×100%
      百分數→小數：÷100
      分數→百分數：先轉小數→再×100%

  必記常用互換（SSPA 超常考！）：
      1/2 = 0.5 = 50%    1/4 = 0.25 = 25%
      3/4 = 0.75 = 75%   1/5 = 0.2 = 20%
      1/8 = 0.125 = 12.5%   1/10 = 0.1 = 10%
      1/3 ≈ 0.333 = 33.3%   2/3 ≈ 0.667 = 66.7%

  口訣：「分轉小=除，小轉百=乘100，百轉小=除100！」

二、核心知識精講 ＋ 例題練習

知識點一：分數 → 小數 → 百分數 🔴

  ① 分數轉小數：分子 ÷ 分母
  ② 小數轉百分數：× 100，加 %
  ③ 除唔盡→通常取三位有效數字

  例題：3/8 轉小數再轉百分數
      3÷8 = 0.375 → ×100 = 37.5% ✅


知識點二：百分數 → 小數 → 分數 🔴

  ① 百分數轉小數：÷ 100
  ② 小數轉分數：寫成十分之幾/百分之幾→約簡

  例題：45% 轉小數再轉分數
      45% = 0.45 = 45/100 = 9/20 ✅
