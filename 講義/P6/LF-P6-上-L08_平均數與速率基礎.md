===== WHY BOX：平均數唔係「中間數」！=====

  好多同學以為：1, 2, 3, 4, 5 嘅平均數 = 3（中間嗰個）
  → 今次啱，但只係巧合！換一組數字即刻錯！
  
  例如：1, 1, 1, 1, 100
     中間數 = 1
     真正平均數 = (1+1+1+1+100)÷5 = 20.8
  
  平均數嘅真正意思：將總數「平均分」畀每個人/每組
  即係：如果所有人都一樣多，每人應該有幾多？
  
  考試平均分例子：
     中文82 + 英文75 + 數學91 + 常識88
     → 總分 = 336，平均 = 336÷4 = 84分
  
  口訣：「先加晒，再平分！總和÷個數 = 平均數」

===== 二、核心知識精講 ＋ 例題練習 =====

--- 知識點一：平均數基本公式 🔴 SSPA ---

平均數 = 總和 ÷ 個數

| 公式變形 | 用途 | 例子 |
|------|------|------|
| 平均 = 總和÷個數 | 求平均 | 總分300，5科→60分 |
| 總和 = 平均×個數 | 已知平均反求總和 | 平均60分，5科→總分300 |
| 個數 = 總和÷平均 | 求有幾多個 | 總分300，平均60→5科 |

例題1：小文五次數學測驗成績：78, 82, 85, 90, 95。求平均分。
解：總分 = 78+82+85+90+95 = 430
    平均 = 430÷5 = 86分

例題2：小美四次英文默書平均分=88。求四次總分。
解：總分 = 88×4 = 352分
    ⚠️ 唔係88！平均分=88代表總分÷4=88！

--- 知識點二：加權平均（唔同權重）🔴 SSPA ---

當數據有「重複」或「權重」時：
  平均 = (數值₁×次數₁ + 數值₂×次數₂ + ...) ÷ 總次數

例題3：一個果籃有蘋果和橙。蘋果3個，每個200g；橙5個，每個150g。
求水果的平均重量。
解：總重量 = 3×200 + 5×150 = 600 + 750 = 1350g
    總數量 = 3+5 = 8個
    平均重量 = 1350÷8 = 168.75g
    
    ⚠️ 唔係 (200+150)÷2 = 175g！因為蘋果同橙嘅數量唔同！

例題4：全班40人，男生25人平均身高145cm，女生15人平均身高138cm。
求全班平均身高。
解：男生總身高 = 25×145 = 3625cm
    女生總身高 = 15×138 = 2070cm
    全班總身高 = 3625+2070 = 5695cm
    全班平均 = 5695÷40 = 142.375 ≈ 142.4cm
    
    ⚠️ 唔係 (145+138)÷2 = 141.5！男女生人數唔同！

--- 知識點三：已知平均反求個別值 🔴 SSPA 殺手級！---

策略：先求總和（平均×個數），再減已知值

例題5：小華五次測驗，前四次分別是：75, 80, 85, 90。
如果五次平均要達到85分，第五次最少要幾多分？
解：需要總分 = 85×5 = 425
    前四次總分 = 75+80+85+90 = 330
    第五次需要 = 425−330 = 95分

例題6（陷阱！）：六個數的平均=30。其中五個數是：28, 32, 25, 35, 27。
求第六個數。
解：六個數總和 = 30×6 = 180
    已知五個總和 = 28+32+25+35+27 = 147
    第六個 = 180−147 = 33
    
    ⚠️ 總和180，唔係30！平均30代表總和=30×6=180！

平均數常與棒形圖/折線圖合併考核

例題7：下面的棒形圖顯示商店四天的銷售額。
星期一：$400，星期二：$350，星期三：$500，星期四：$450。
(a) 求四天平均銷售額。
(b) 老闆目標是五天平均達到$460。星期五最少要賣多少？

         平均 = 1700÷4 = $425

         星期五需要 = 2300−1700 = $600

===== 五、課堂總結 + 必記口訣（3 分鐘）=====

📌 平均數 = 總和 ÷ 個數（唔係中間數！）
📌 總和 = 平均 × 個數（反求必用！）
📌 加權平均：先各乘權重，加總再除總權重
📌 缺項題：先求總和（平均×個數）→ 減已知 → 得未知
📌 口訣：「平均=總和÷個數，總和=平均×個數！」
📌 最致命：加權平均唔係「平均嘅平均」！
