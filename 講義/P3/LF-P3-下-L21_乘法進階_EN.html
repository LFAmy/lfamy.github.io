二、Core Knowledge （15 min） ＋ Worked Examples
知識點一：三位數 x 一位數 — 直式基本功⭐ 最重要
① 步驟：個位 → 十位 → 百位，逐位乘！
② 123 x 3：3x3=9（個位），2x3=6（十位），1x3=3（百位）→ 369
③ 關鍵：每位乘完如果 ≥10，就要進位！
④ Mnemonic：「乘完寫個位，十位進上去，下一輪要加返！」
例題 1：冇進位（最簡單）
🍬 每盒有 132 粒糖，買 3 盒共有幾粒？
直式：132 x 3
個位：2x3=6 | 十位：3x3=9 | 百位：1x3=3
答案：396 粒
🪤 例題 2：有進位！（Trap重點）
📦 每箱有 156 個橙，買 4 箱共有幾個？
❌ 小Trap怪獸話
156 x 4 = 424
6x4=24（寫4進2），5x4=20，但忘記加進位2！
✅ 正確答案
156 x 4 = 624
6x4=24（寫4進2）；5x4=20，20+2=22（寫2進2）；1x4=4，4+2=6；答案=624
⚠️ 致命Trap：進位一定要加返！每次乘完之後，要加返低位進上嚟嘅數字！呢個係 P3 學生最容易錯嘅位！
💡 小貼士：用螢光筆圈住進位數字，提醒自己一定要加返！

知識點二：有「0」嘅Multiplication🔴 必考
① 中間有 0：105 x 3 = 315（0x3=0，唔可以漏！）
② 尾有 0：240 x 3 = 720（先計 24x3=72，再加返個 0）
③ Trap：105 x 3，十位係0，但百位1x3=3要寫啱位！唔可以寫成 315？錯！係 315 ✓
④ 祕技：尾有0 → 先當冇0咁乘，最後加返0！
🪤 例題 3：中間有 0 嘅真正Trap
108 x 5 = ？（呢題先真正考到你！）
❌ 小Trap怪獸話
108 x 5 = 500
跳過咗十位嘅 0！直接由 1x5=5，再計 8x5=40，寫成 500。漏咗 0x5=0 呢步，對位錯晒！
✅ 正確答案
108 x 5 = 540
逐位乘：8x5=40（寫0進4）；0x5=0，0+4=4（寫4進0）；1x5=5。答案=540。每位都要乘，0嗰位都唔可以跳！
⚠️ 中間有 0 嘅Trap：唔可以跳過任何位！就算係 0 都要乘（0 × 任何數 = 0），然後加返進位。跳咗一步就對錯位！
⚠️ 重點：每次乘完要確認寫喺邊個位（個/十/百），有0嗰位都要乘，結果係0都要寫！

知識點三：Estimation驗證法⭐
① 做完Multiplication，用Estimation檢查答案是否合理！
② 例：287 x 3 ≈ 300 x 3 = 900，真正答案應該在 900 附近
③ 如果計到 621 或 1287，就知一定出錯！
④ EstimationMnemonic：「四捨五入變整數，乘完之後比較下！」

#Common ErrorCorrect Approach
1進位忘記加返：乘完冇加低位進上嚟嘅數每次乘完後加返進位數字，用螢光筆圈住進位！
2中間有0漏咗：105x3=315 計成 135每位都要乘，0x3=0，唔可以跳過！
3尾有0當冇0：240x3 計成 72先計24x3=72，再加返個0 → 720
4進位寫錯位：進到十位但寫咗喺百位進位要寫喺正確嘅位（上一位）
5冇做Estimation驗證心算約數檢查答案是否合理
6乘數次序亂咗：由百位開始乘必定由個位開始乘，逐位上！
7直式對位唔齊個位對個位、十位對十位，用間尺畫線！
🧠 Mnemonic一：「個位乘起逐位上，乘完加返進位數；中間有零唔好跳，尾有零就最後補！」
🧠 Mnemonic二：「乘完要驗證，Estimation幫到手；約數乘一乘，答案心中有！」
