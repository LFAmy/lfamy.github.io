## 3. 六個競爭場景模擬

### 場景 1：大型社發動價格戰 🔴🔴

**條件**：遵理/現代推出「首月 $99 試讀 + 第二個月半價」，廣告鋪天蓋地

| 畫像群 | 反應 | 霖楓流失風險 | 對策 |
|--------|------|------------|------|
| A1 精英媽 | 「便宜沒好貨」— 不受影響 | 低 | 強調「小班 vs 大班錄播」差異 |
| A2 虎媽 | 可能試試，但要求高 → 不滿意就回來 | 中 | 保留名額，追蹤離開的家長 |
| B1 跟風媽 | **首批流失** — 「大家都去了」 | **高** | WhatsApp 群組發布真實對比 |
| B2 務實型 | 計算後「試堂成本低」→ 可能試 | 中 | 提供免費學習評估做差異化 |
| B3 忙碌型 | 看方便性 → 若地點不便不換 | 低 | 保持服務品質 |
| B4 新移民 | **最敏感** — 預算緊 → 可能轉 | **高** | 推出「2 人同行優惠」 |

**模擬結果**：30 天內流失 15-25% 潛在客戶，主要為 B1/B4 群。60 天後部分回流（大型社服務不如預期）。

### 場景 2：同區小型社開業 🟡

**條件**：同區新開一家小型數學專科社，主打「前名校教師」

| 畫像群 | 反應 | 對策 |
|--------|------|------|
| A 群 | 比較兩家 → 看實際教學品質 | 展示 70+ 自製教材的差異化 |
| B1 | 兩家都追蹤 → 看誰先給「感覺」 | 先接觸先贏 → 加速首次接觸到試堂的時間 |
| B2 | 比較價格和距離 | 若價格相同，教材優勢是關鍵 |

**模擬結果**：若對手有「名校教師」光環，霖楓需用**自製教材可視化**（MathJax/SVG 動畫講義）做差異化對抗。純靠口碑不足以抗衡「前名校教師」標籤。

### 場景 3：學校推出免費課後輔導 🟡🟡

**條件**：目標學校獲基金資助，提供免費數學輔導班

| 影響 | 說明 |
|------|------|
| 短期 | B2/C2 群直接流失（免費 vs 付費） |
| 中期 | 免費班品質不如預期 → 部分回流 |
| 長期 | 學校輔導無法提供個性化 → 霖楓定位「補充學校不足」 |

**對策**：與學校合作，成為「學校推薦的校外進階選擇」，而非競爭關係。

### 場景 4：經濟衰退加深 🔴🔴🔴

**條件**：香港經濟持續下行，家庭削減非必需開支

| 畫像群 | 行為 | 影響 |
|--------|------|------|
| A 群 | 幾乎不影響（教育支出有慣性） | 低 |
| B2/B4 | **削減或降級**（從每週 2 堂→1 堂） | 中高 |
| C1/C2 | **完全退出市場** | 高（但本來就不是目標） |

**模擬**：總需求萎縮 20-30%，但高端市場（A 群）需求穩定。**策略：向上移動定價，做「精品小班」定位，放棄低端價格戰。**

### 場景 5：競爭對手爆出負面新聞 🟢🟢

**條件**：大型社爆出「導師造假學歷」或「錄播充直播」醜聞

| 影響 | 說明 |
|------|------|
| 即時 | 家長對「大型社」信任受損 |
| 1-2 週 | A/B 群開始尋找替代方案 |
| 機會窗口 | 2-4 週內加速展示霖楓的「真實教學」差異 |

**策略**：預先準備「透明教學」內容（導師真名、真實課堂片段、學生進步數據），在行業危機時立即推送。

### 場景 6：教育局新規加強補習社監管 🟡

**條件**：廣告限制、導師資格強制登記、班額上限

| 影響 | 說明 |
|------|------|
| 對大型社 | 廣告受限、導師資格審查 → 成本上升 |
| 對霖楓 | 班額限制 → 若已小班教學則不受影響 |
| 機會 | 行業洗牌，不合格者退出 |

- 霖楓維持現有教學品質和教材優勢
