Google Sheets 雲端後端設定

跟住做 · 5 分鐘完成 · 跨裝置同步

第 1 步：建立 Google Sheets

1

建立新試算表

打開 sheets.google.com → 點「+ 空白」→ 改名為 霖楓CRM

✅ 見到全新空白試算表

第 2 步：打開 Apps Script

2

打開程式碼編輯器

Sheets 選單：擴充功能 → Apps Script

✅ 見到新分頁，內有 function myFunction() { }

第 3 步：貼上後端代碼

3

複製並貼上

打開你電腦上嘅：
G:\lam-fung-academy\scripts\cloud-backend.gs

全選（Ctrl+A）→ 複製（Ctrl+C）→ 去 Apps Script 編輯器 → 刪除原有代碼 → 貼上（Ctrl+V）→ 儲存（💾）

⚠️ 確保代碼全部貼上，不要遺漏任何一行

第 4 步：初始化

4

執行 testSetup

1. 下拉選單揀 testSetup
2. 按 ▶️ 執行
3. 授權（揀你嘅 Google 帳戶 → 允許）

✅ 返去 Sheets 見到 7 個新分頁

第 5 步：部署

5

部署為 Web App

右上角 部署 → 新增部署 → 類型揀 網路應用程式 → 存取權限揀 任何人 → 按 部署 → 複製 URL

⚠️ 必須保存此 URL！格式：https://script.google.com/macros/s/XXXXX/exec

第 6 步：連接平台

6

貼上 URL

打開 admin-cloud.html → 貼上 URL → 儲存並連接 → 見到「已連接 ✅」

✅ 從此所有數據自動同步到 Google Sheets
打開雲端管理中心 返回 Launchpad
