追蹤系統啟動指南

1 個 FB Pixel ID + 1 個 GA4 ID = 全站追蹤即時上線

你只需要 2 個 ID

ID去邊度搵狀態
FB Pixel ID（16位數字） FB 事件管理工具 → 選擇 Pixel → 設定 等你提供
GA4 Measurement ID（G-開頭） Google Analytics → 管理 → 資料串流 → 選擇串流 等你提供

第 1 步：搵 FB Pixel ID

  1. 去 business.facebook.com/events_manager2
  2. 左手邊選擇你嘅 Pixel（如果你得一個，會直接顯示）
  3. 左手邊「設定」tab
  4. 你會見到一串 16 位數字 — 呢個就係你嘅 Pixel ID
  5. 複製佢
如果你從來未建立過 Pixel：點「+ 建立」→ 按照指示建立一個新嘅 Pixel。2 分鐘搞掂。

第 2 步：搵 GA4 Measurement ID

  1. 去 analytics.google.com
  2. 左下角「管理」(⚙️)
  3. 「資料串流」→ 選擇你網站嘅串流
  4. 你會見到「Measurement ID」— 格式係 G-XXXXXXXXXX
  5. 複製佢
如果你未設定過 GA4：需要先建立一個 GA4 資源。去 admin → 建立資源 → 跟指示做。5 分鐘搞掂。

第 3 步：俾 ID 我，我幫你全站替換

將你搵到嘅兩個 ID send 俾我，格式：

FB Pixel: 1234567890123456
GA4: G-ABC123DEF4

我會即刻執行：

python scripts/activate_tracking.py --fb-pixel 1234567890123456 --ga4 G-ABC123DEF4

30 秒內，以下全部檔案嘅 placeholder 會被替換：

檔案替換內容
index.htmlFB Pixel ID + GA4 ID
docs/track.htmlFB Pixel ID

啟動後你會得到咩
