正式營運清單

從「系統就緒」→「正式營運」— 逐步執行

0/24 完成

Phase 1: 基礎建設 本週完成

部署 Google Sheets 後端 (script.google.com → 貼上 cloud-backend.gs)
設定雲端 URL (admin-cloud.html → 貼上 Web App URL)
設定 Gitee 鏡像 (gitee.com → 導入倉庫 → 開啟 Pages)
測試國內訪問速度（請內地老師打開 lfamy.gitee.io）
開通微信公眾號（用於內地家長通知）

Phase 2: 內容發布 本週完成

發布 FB 廣告 → landing.html（用 Phase 24 著陸頁）
發布 IG 帖子（用 social_calendar_v2.html 內容）
發布小紅書內容（用 xhs 模板）
設定 Facebook Pixel（tracking-setup.html）
設定百度統計（內地流量追蹤）

Phase 3: 學生激活 下週完成

導入第一批學生（admin-activate-v2.html → 批量生成激活碼）
每位學生做一次陷阱診斷（trap-quiz.html）
用 AI 智能組卷生成第一堂教材（smart-test-gen.html）
設定微信/WhatsApp 通知模板

Phase 4: 第一堂課 下週完成

用教師入職精靈設定課堂（teacher-onboarding.html）
開 QR 簽到（qr-checkin.html → 學生掃碼）
開啟課堂監控台（teacher-class-monitor.html）
落堂 → 生成摘要 → 發送家長

Phase 5: 增長營運 持續

開啟推薦獎勵計劃（referral-rewards.html）
每日檢查營運儀表板（ops-dashboard.html）
每週檢查自動通訊中心（auto-comms.html）
處理待審批付款（admin-approvals.html）
跟進舊線索再激活（reactivation-kit.html）
持續擴充題庫（定期跑 build_mega_question_bank.py）
預計時間線：
Phase 1-2: 1 星期內完成 · Phase 3-4: 第 2 星期完成 · Phase 5: 持續營運
全部完成後，平台進入正式營運狀態。
