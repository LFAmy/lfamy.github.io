🏫 S1中學數學銜接課程

P6升S1係數學最易脫節嘅階段。小學嘅「計答案」變成中學嘅「證明過程」，好多學生適應唔到。

⚠ 小學升中學嘅三大斷層

  • 負數概念：小學從未接觸負數，S1一開學就要做負數四則運算
  • 代數思維：由數字計算轉為代數式化簡，用字母代表數字
  • 證明要求：中學要求寫步驟、畫圖證明，唔再係淨寫答案

📋 S1銜接課程內容（8堂）

  • 第1-2堂：負數概念 + 數線 + 絕對值
  • 第3-4堂：代數式認識 + 同類項合併 + 簡易方程進階
  • 第5-6堂：不等式入門 + 坐標系進階
  • 第7-8堂：S1面積體積公式擴展 + 綜合模擬

🏅 奧數競賽培訓課程

奧數唔係「更難嘅數學」，而係「用唔同方法思考數學」。訓練邏輯推理、創意解難、競賽策略。

🎯 適合邊個？

  • 校內數學成績優異（85分以上），想更上一層樓
  • 對數學有濃厚興趣，喜歡挑戰難題
  • 準備參加華夏盃、港澳盃、IMO等數學競賽
  • 想提升邏輯思維，為中學理科鋪路

📋 課程內容（12堂一循環）

  • 數論入門：質數、整除性、同餘、最大公因數應用
  • 組合數學：排列組合、鴿巢原理、邏輯推理
  • 幾何思維：輔助線技巧、面積法、對稱應用
  • 競賽策略：時間管理、選擇題技巧、逆向思維
  • 真題實戰：歷屆華夏盃、港澳盃真題分析

👑 精英學校備戰課程

針對直資/私立名校（DBS、SPCC、DGJS等）嘅數學筆試同面試，包括英文數學題。

🏛 目標學校類型

  • 直資學校：DBS、SPCC、協恩、英華等（中英數學卷）
  • 私立名校：DGJS、聖保羅男女等（英文數學）
  • 官津名校：皇仁、喇沙、瑪利諾等（自行收生面試）

📋 課程特點

  • 英文數學訓練：熟悉英文數學題型同詞彙（fraction、decimal、percentage）
  • 面試模擬：數學邏輯題口試練習、即時解題展示
  • 超前學習：覆蓋S1部分內容，令學生在筆試中脫穎而出
  • 個人化計劃：根據目標學校設計專屬備戰方案
