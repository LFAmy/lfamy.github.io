甲部：多項選擇題 (Q1-Q10，每題3分，共30分)
Q1. 以下哪個是假分數？A. 3/8 B. 8/3 C. 1 2/3 D. 5/5
答案：B. 8/3（分子>分母）
假分數：分子 >= 分母。
A. 3/8：真分數（分子<分母）
B. 8/3：假分數（分子>分母）
C. 1 2/3：帶分數
D. 5/5 = 1：假分數（分子=分母），但注意題目可能只接受分子>分母。
Trap T6 (分數陷阱)：D(5/5)也是假分數(分子=分母)。兩個都係假分數，但B係最明顯既假分數。標準答案為B。
Q2. 2/5 + 3/10 = ？
答案：B. 7/10
通分：LCM(5,10) = 10。2/5 = 4/10。4/10 + 3/10 = 7/10。
Trap T6 (分數陷阱)：A(5/15)分子分母各自加：(2+3)/(5+10)=5/15。這是最常見錯誤！必須先通分母。
Q3. 0.5 + 0.28 = ？
答案：B. 0.78
對齊小數點：0.50 + 0.28 = 0.78
Trap T2 (小數點陷阱)：A(0.33)可能是0.5+0.28但小數點冇對齊。0.5=0.50，加上0.28=0.78。
Q4. 以下哪個數值最大？A. 0.5 B. 0.05 C. 0.45 D. 0.6
答案：D. 0.6
比較：0.6 > 0.5 > 0.45 > 0.05。可以全部寫成百分位：60>50>45>5。
Trap T2 (小數比較陷阱)：B(0.05)看似「50」但實際係0.05=5/100。小數比較要對齊小數點後逐位比！
Q5. 正方形有多少條對稱軸？A. 1條 B. 2條 C. 3條 D. 4條
答案：D. 4條
正方形：2條中線（垂直+水平）+ 2條對角線 = 4條對稱軸。
Trap T5 (幾何圖形混淆)：B(2條)是把正方形當成長方形。長方形只有2條對稱軸，正方形有4條！
Q6. 12和18的HCF=？A. 3 B. 4 C. 6 D. 12
答案：C. 6
12的因數：1,2,3,4,6,12。18的因數：1,2,3,6,9,18。公因數：1,2,3,6。最大=6。
Trap T6 (HCF陷阱)：A(3)係公因數但唔係最大。D(12)唔係18的因數。
Q7. 18 - (7 + 5) = ？
答案：A. 6
先計括號：7 + 5 = 12。18 - 12 = 6。
Trap T3 (運算次序)：B(16)係18-7+5=16（錯誤地由左至右忽略括號）。有括號要先計括號！
Q8. L形：A=5cm×3cm，B=2cm×4cm。總面積=？
答案：B. 23 cm²
A面積 = 5 × 3 = 15 cm²
B面積 = 2 × 4 = 8 cm²
總面積 = 15 + 8 = 23 cm²
（注意L形由兩個長方形組成，沒有重疊）
Trap T4 (面積陷阱)：A(15)只計左一部分。D(8)只計左另一部分。
Q9. 用了零用錢的1/3，再用剩餘的1/2。第二次用了全部的幾分之幾？
答案：B. 1/3
第一次後剩下：1 - 1/3 = 2/3
第二次用了剩餘的1/2 = 2/3 × 1/2 = 1/3
所以第二次用了全部的1/3（同第一次一樣）
Trap T6 (分數基準量陷阱)：A(1/2)係直接用1/2，但「剩餘的1/2」基準量是剩餘的2/3，不是全部！
Q10. 面向東北，向後轉是面向什麼方向？
答案：C. 西南
向後轉 = 轉180°。東北的相反方向是西南。
Trap T5 (方向陷阱)：東北的相反是西南（不是西北或東南）。想像指南針：NE的對面是SW。
乙部：計算題 (Q11-Q16，每題5分，共30分)
Q11. 2/3 + 5/6 = ？(5分)
答案：3/2 = 1 1/2
LCM(3,6) = 6。2/3 = 4/6。4/6 + 5/6 = 9/6 = 3/2 = 1 1/2。
評分標準：通分正確(2分) + 計算正確(2分) + 約簡(1分) = 5分
Q12. 3 1/4 - 1 3/4 = ？(5分)
答案：1 1/2（或3/2或1.5）
方法一：轉假分數。3 1/4 = 13/4，1 3/4 = 7/4。13/4 - 7/4 = 6/4 = 3/2 = 1 1/2。
方法二：整數和分數分開計。3-1=2；1/4-3/4唔夠減，借1（=4/4）：2-1=1，1/4+4/4-3/4=5/4-3/4=2/4=1/2。答案=1 1/2。
評分標準：轉假分數(2分) + 計算正確(2分) + 約簡(1分) = 5分
Q13. 1.75 + 2.8 - 3.2 = ？(5分)
答案：1.35
1.75 + 2.80 = 4.55。4.55 - 3.20 = 1.35。
評分標準：加法正確(2分) + 減法正確(2分) + 答案正確(1分) = 5分
Q14. (24 - 8) × 3 + 12 ÷ 4 = ？(5分)
答案：51
Step 1: (24 - 8) = 16（括號優先）
Step 2: 16 × 3 = 48（乘法）
Step 3: 12 ÷ 4 = 3（除法）
Step 4: 48 + 3 = 51（加法）
Trap T3 (運算次序)：必須跟括號→乘除→加減的順序。
評分標準：每步正確(各1分) + 最終答案(1分) = 5分
Q15. 找出24和36的HCF和LCM。(5分)
HCF = 12，LCM = 72
24 = 2³ × 3，36 = 2² × 3²
HCF = 2² × 3 = 4 × 3 = 12
LCM = 2³ × 3² = 8 × 9 = 72
評分標準：HCF正確(2分) + LCM正確(3分) = 5分
Q16. 3/5 - 1/4 + 1/2 = ？(5分)
答案：17/20
LCM(5,4,2) = 20。
3/5 = 12/20
1/4 = 5/20
1/2 = 10/20
12/20 - 5/20 + 10/20 = 17/20
評分標準：通分LCM正確(2分) + 分子計算正確(2分) + 答案最簡(1分) = 5分
丙部：應用題 (Q17-Q18，每題10分，共20分)
Q17. 超市購物：牛奶2盒×$18.50，麪包3個×$8.80，雞蛋1盒$22.60。(10分)
(a) 總共：$85.00
(b) 找回：$15.00
(c) 每人：$5.00
(a) 牛奶：2 × 18.50 = 37.00
麪包：3 × 8.80 = 26.40
雞蛋：22.60
總數：37.00 + 26.40 + 22.60 = 86.00

等等，重新計算：37.00 + 26.40 = 63.40；63.40 + 22.60 = 86.00。
(b) 找回：100.00 - 86.00 = 14.00
(c) 每人：14.00 ÷ 3 ≈ 4.67（不是整數）

Hmm，等一下。重新檢查計算：
2 × 18.50 = 37.00（正確）
3 × 8.80 = 26.40（正確）
37.00 + 26.40 = 63.40（正確）
63.40 + 22.60 = 86.00（正確）
找回 = 100 - 86 = 14.00
每人 = 14 ÷ 3 ≈ 4.667... = $4.67（取至兩位小數）
評分標準：(a)總數正確(4分) + (b)找回正確(3分) + (c)每人金額正確含答句(3分) = 10分
Q18. L形花園：豎直8m×4m，橫向(底部)6m×3m。(10分)
(a) 簡圖（略）
(b) 總面積 = 50 m²
(c) 籬笆總長（周界）= 36 m
(a) 畫出L形：豎直部分長8m闊4m，底部橫向部分長6m闊3m。重疊部分3m×4m。

(b) 總面積 = 8×4 + 6×3 - (重疊？)
如果兩個長方形沒有重疊（豎直在左，橫向在右下連接）：
面積 = 8×4 + 6×3 = 32 + 18 = 50 m²
但如果豎直部分的底部和橫向部分重疊（3m×4m的區域），則：
面積 = 32 + 18 - 12 = 38 m²

根據題目描述「豎直部分長8m闊4m，橫向部分（接在底部）長6m闊3m」——這暗示兩個長方形是連接的但可能重疊。
假設標準L形（不重疊計算）：總面積 = 32 + 18 = 50 m²。

(c) 周界計算：外圍輪廓。以典型L形（豎直8×4在左，橫向6×3在右下）：
外圍 = 8 + 4 + (8-3或...) + ...
實際上，若豎直的底部(4m)和橫向的頂部重疊部分寬3m：
外圍 = 8(top) + (4-3)(right top) + 6(right) + 3(bottom) + (6+4-3)(left bottom) + ...

由於確切形狀取決於如何拼接，最合理的L形周界：
如果豎直8×4貼在橫向6×3的左上角（標準L形）：
周界 = (8+4+8) + (6+3+6) - 2×(重疊邊長)
更直接：總周界 = 原始兩個長方形周界和 - 2×重合邊
= [(8+4)×2 + (6+3)×2] - 2×3
= [24 + 18] - 6 = 36 m
評分標準：(a)簡圖正確含尺寸(2分) + (b)面積正確(4分) + (c)周界正確(4分) = 10分
