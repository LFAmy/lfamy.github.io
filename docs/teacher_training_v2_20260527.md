# 霖楓學苑 · 新教師培訓系統 v2.0
## 「三日成功上課」密集培訓計劃
### 總時數：3天 × 1.5小時 = 4.5小時 | 每堂65分鐘

---

## 培訓理念

> **「不是培訓老師如何教數學，而是培訓老師如何讓學生愛上數學。」**

霖楓學苑的核心差異化：**陷阱診斷教學法 + 香港在地化 + 一對三互動模式**。
新老師不需要是最聰明的數學家，但必須是最能讀懂學生錯誤、最能製造課堂歡樂的人。

---

## Day 1：基本功 — 香港數學教學的DNA（1.5小時）

### Section 1.1：香港小學數學全景圖（20分鐘）

**目標：** 讓老師理解香港數學教育的整體脈絡

| 時間 | 內容 | 形式 |
|------|------|------|
| 0-5min | 香港教育局數學課程架構簡介（P1-P6三大範疇：數、度量、圖形與空間、數據處理） | 圖解簡報 |
| 5-10min | **SSPA呈分試解密**：呈分試如何影響升中派位？數學科的戰略地位 | 真實數據+案例 |
| 10-15min | 香港家長心態分析：「分數就是一切」→ 如何轉化為霖楓的價值主張 | 角色扮演 |
| 15-20min | P3-P6年級知識點地圖：霖楓40堂大綱如何覆蓋全部呈分試重點 | 互動問答 |

**核心金句：** 「家長花錢不是買數學知識，是買呈分試的分數和升中的安心。」

### Section 1.2：香港用字與解題方法（25分鐘）

**目標：** 確保老師使用純正香港數學語言

| 時間 | 內容 | 形式 |
|------|------|------|
| 0-8min | **香港 vs 內地 vs 台灣 用字對照表**（必考！） | 快速測驗 |
| 8-15min | 關鍵用字：小數點（非「小数点」）、括號（非「括号」）、闊度（非「寬度」）、周界（非「周長」）、速率（非「速度」） | 配對遊戲 |
| 15-20min | **香港特有解題步驟**：逐一步驟展示（例：面積題→「寫公式→代數字→計答案→寫單位→寫答句」五步法） | 示範+練習 |
| 20-25min | 常見香港課本題型辨認：現代教育研究社 vs 朗文 vs 牛津 | 實物展示 |

**香港用字速記卡（10個最易錯）：**
1. 周界（非周長/ perimeter）
2. 闊度/闊（非寬度/寬）
3. 括號（非括弧/括號）
4. 速率（非速度 - P6才區分）
5. 小數點（非小數點 - 但讀法不同）
6. 角（非角度 - 「角度」是指角的大小）
7. 梯形（非梯形 - 香港也有用梯形）
8. 棒形圖（非條形圖/柱狀圖）
9. 方程（非方程式 - 小學用方程）
10. 答句（非答案句/回答）

### Section 1.3：65分鐘課堂結構拆解（20分鐘）

**目標：** 讓老師完全掌握霖楓65分鐘課堂的標準流程

```
霖楓標準65分鐘課堂結構：
┌─────────────────────────────────────────┐
│ 0-3min   🎯 破冰暖身（故事/笑話/魔術開場）    │
│ 3-8min   📋 上堂回顧 + 本堂目標展示          │
│ 8-20min  🧠 核心概念講解（陷阱前置標註！）    │
│ 20-25min ✏️ 例題示範（香港五步法）            │
│ 25-40min 🎮 互動練習（一對三輪流答題）        │
│ 40-50min ⚡ 陷阱揭示 + 口訣記憶               │
│ 50-60min 📝 獨立練習 + 個別指導              │
│ 60-65min 🏆 總結 + 金句 + 下堂預告           │
└─────────────────────────────────────────┘
```

**關鍵時間節點訓練：**
- 老師必須在8分鐘內完成暖身+回顧+目標（不超時）
- 核心概念講解不超過12分鐘（避免「講太多」）
- 互動練習是課堂靈魂，必須佔15分鐘以上
- 陷阱揭示必須讓學生「哇」一聲

### Day 1 課後作業：
- 背熟10個香港用字對照
- 錄製一段3分鐘「破冰開場」（可選任何課題）
- 閱讀指定P5講義L01-L05

## Day 2：教學力 — 陷阱診斷法與互動設計（1.5小時）

### Section 2.1：陷阱診斷教學法深度訓練（30分鐘）

**目標：** 這是霖楓最核心的教學法，老師必須內化

**陷阱診斷四步法：**

| 步驟 | 名稱 | 說明 | 例子（三角形面積） |
|------|------|------|-------------------|
| ① | **預告陷阱** | 教新概念前，先告訴學生「這裡有個坑」 | 「今日我教你一個全世界最多人錯嘅嘢...」|
| ② | **故意出錯** | 老師示範常見錯誤，讓學生「捉錯處」 | 三角形面積忘記÷2，全班一齊大叫「錯！」 |
| ③ | **口訣固化** | 把正確做法變成口訣，有節奏地唸 | 「底乘高，除以二；唔除二，就玩完！」 |
| ④ | **陷阱標記** | 講義上用🌿🌳標記陷阱難度 | 🌿=常見錯 🌳=SSPA高危 💀=終極陷阱 |

**實戰演練（15分鐘）：**
- 每位新老師抽取一個課題，即場設計陷阱教學
- 題目池：平行四邊形面積、異分母加法、小數乘法、方程
- 同儕互相「扮演學生」，故意犯錯測試老師

### Section 2.2：一對三互動設計（25分鐘）

**目標：** 掌握一對三模式的獨特教學技巧

**一對三黃金法則：**

| 法則 | 說明 | 操作 |
|------|------|------|
| **輪流制** | 每個學生都有發言機會 | 用計時器，每2分鐘換人 |
| **競爭制** | 三人互相競爭但不傷和氣 | 計分板 + 星星獎勵 |
| **互助制** | 會的教不會的 | 「小明你教下小美啦」 |
| **點名制** | 隨機點名回答，保持專注 | 用轉盤/骰子隨機選人 |

**互動技巧工具箱：**
1. **「你覺得呢？」** — 永遠先問學生想法，不要直接給答案
2. **「錯得好！」** — 學生答錯時的反應：先讚賞勇氣，再引導思考
3. **「試下另一個方法」** — 鼓勵一題多解
4. **「三個人都寫完先揭曉」** — 防止互相抄襲
5. **「你做老師教我」** — 讓學生反過來教老師（最強學習法）

### Section 2.3：趣味性培訓（20分鐘）

**目標：** 數學課可以很好玩！老師要學會「表演」

**故事化教學技巧：**

| 課題 | 故事/比喻 | 效果 |
|------|----------|------|
| 平行四邊形面積 | 「平行四邊形係一個企歪咗嘅長方形！」 | 直觀理解 |
| 分數通分 | 「兩個唔同國家嘅人，要講同一種語言先可以傾計」 | 易記 |
| 代數x | 「x就係一個神秘人，我哋要搵出佢係邊個！」 | 好奇心 |
| 三角形面積÷2 | 「兩個一模一樣嘅三角形可以拍埋變長方形，所以÷2」 | 視覺化 |
| 體積 | 「面積係地板，體積係成間房！」 | 2D→3D |

**課堂氣氛掌控：**
- 「3分鐘笑一次」原則：每3分鐘要有一個笑點/驚喜
- 聲音變化：重點要大聲、陷阱要細聲、故事要有起伏
- 肢體語言：面積用雙手畫、分數用手指切、方程用身體做等號

### Day 2 課後作業：
- 為P5 L03（平行四邊形與三角形面積）設計完整陷阱教案
- 錄製一段5分鐘的「帶陷阱的例題示範」
- 設計3個數學小遊戲

## Day 3：實戰力 — 全真模擬與家長溝通（1.5小時）

### Section 3.1：全真模擬授課（35分鐘）

**目標：** 在真實壓力下完成一堂完整65分鐘課

**模擬設置：**
- 2位培訓師扮演「學生」（一個聰明、一個常錯、一個發呆）
- 1位扮演「家長」（在旁邊觀察）
- 新老師完整執行65分鐘課堂

**評分標準（100分）：**

| 評分項目 | 佔比 | 優秀標準 |
|---------|------|---------|
| 破冰開場（3min） | 10% | 學生笑咗就贏 |
| 核心概念清晰度 | 25% | 三個學生都講得出重點 |
| 陷阱標記準確度 | 20% | 學生識得自己避開陷阱 |
| 互動均衡度 | 15% | 三個學生都有同等發言 |
| 時間控制 | 15% | 65分鐘±3分鐘完成 |
| 趣味性/親切度 | 15% | 學生唔捨得落堂 |

### Section 3.2：模擬檢討與優化（15分鐘）

**檢討流程：**
1. 培訓師回饋（5分鐘）：直接點出3個亮點+3個改善點
2. 同儕回饋（5分鐘）：其他新老師給予具體建議
3. 自我反思（5分鐘）：寫下「如果再教一次，我會...」

### Section 3.3：教師親切度與家長溝通（25分鐘）

**目標：** 老師不止教學生，還要讓家長放心

**教師親切度提升技巧：**

| 技巧 | 操作 | 為什麼有效 |
|------|------|----------|
| **叫名唔叫花名** | 記住每個學生的名字，堂上多用 | 學生感覺被重視 |
| **「我細個都錯過」** | 分享自己小時候犯的錯 | 縮短師生距離 |
| **具體讚賞** | 「你今次寫步驟好整齊」而非「你好叻」 | 學生知道什麼做對了 |
| **課後一句話** | 落堂後對每個學生說一句個人化的話 | 建立情感連結 |
| **記住學生的事** | 上次說過的興趣/考試，今堂問返 | 顯示真心關心 |

**家長溝通劇本（5分鐘版）：**

```
老師：「你好呀，XX媽媽/爸爸！今日XX上堂好投入呀，
      佢喺（具體課題）方面進步咗好多。
      特別係（具體亮點），之前成日錯嘅（陷阱名稱），
      今日佢自己識得避開喇！
      下一步我哋會重點幫佢突破（下一個目標）。
      你有冇咩想特別同我講嘅？」

關鍵原則：
✅ 先讚後建議（永遠不要一開始就投訴）
✅ 具體不空泛（不要只說「佢好乖」）
✅ 給下一步希望（不要只說問題不給方案）
✅ 30秒內講完重點（家長很忙）
```

### Section 3.4：模擬家長溝通（10分鐘）

**角色扮演：**
- 培訓師扮演不同類型的家長
  - Type A：「佢成績跌咗好多喎，點解㗎？」（焦慮型）
  - Type B：「佢話上堂好悶喎」（投訴型）
  - Type C：「可唔可以畀多啲練習佢做？」（雞血型）
- 新老師即場應對

### Section 3.5：畢業認證與持續發展（5分鐘）

**畢業條件：**
- ✅ 模擬授課達70分以上
- ✅ 香港用字測驗100%正確
- ✅ 能背誦霖楓十大教學金句
- ✅ 完成一個課題的完整陷阱教案

**持續發展路徑：**
- Week 1-2：影子教學（觀看資深老師上課）
- Week 3-4：輔助教學（負責課堂練習環節）
- Week 5+：獨立授課（每週一次教學檢討）

## 霖楓十大教學金句（老師必背）

1. 「錯唔緊要，錯完識改先係學習！」
2. 「呢個位全香港最多人錯，你唔好中招呀！」
3. 「數學唔係背，係明！明咗一世都記得。」
4. 「你覺得呢？」（永遠先問）
5. 「試下用另一個方法諗？」
6. 「你做老師教我一次！」
7. 「寫步驟！步驟就係你嘅思考痕跡。」
8. 「口訣要唸出嚟，唔係睇！」
9. 「三個人都寫完先揭曉答案～」
10. 「今日你學到嘅最重要一樣嘢係咩？」

## 附錄B：課堂應急處理手冊

| 情況 | 應對方案 |
|------|---------|
| 學生完全聽不懂 | 退回上一步，換另一種比喻，不要重複同樣解釋 |
| 學生吵架 | 暫停教學，冷靜處理：「我哋係一個團隊，一齊解決問題」 |
| 時間不夠 | 果斷跳過非核心練習，確保陷阱+總結環節 |
| 時間太多 | 加插「挑戰題」，或讓學生互相出題考對方 |
| 家長在旁邊干擾 | 微笑說：「我落堂之後再同你詳細傾，而家先俾小朋友專心」 |
| 網絡/技術問題 | 預先準備離線版講義PDF，隨時可用WhatsApp繼續 |
