📖 霖楓學苑 · 完整使用指南

從講義生產到網站部署，從招生行銷到教學營運，全部功能一次搞懂。

🚀 3分鐘快速開始

1
打開總控制台：在瀏覽器打開 G:\lam-fung-academy\launchpad.html
（或網上版：https://lfamy.github.io/launchpad.html）
2
瀏覽講義：點擊「講義總索引」→ 搜尋或按年級瀏覽 208 份講義
3
生成答案冊：VS Code 終端機貼上：
python scripts/auto_answer_key.py --grade P5
4
發送家長報告：打開「家長報告生成器」→ 填寫 → 一鍵複製到 WhatsApp
5
部署到網站：終端機貼上：
python scripts/deploy.py --push

📚 講義系統

如何查找講義

1
打開 講義/master_index.html（可搜尋關鍵詞：如「面積」「分數」）
2
按年級瀏覽：P3/P4/P5/P6 各40+堂
3
點擊任何講義 → 瀏覽器打開 → Ctrl+P 列印 PDF

講義結構（每份包含）

區塊內容
封面Logo·課題·日期·SSPA關聯·前置知識·目標·陷阱
一、熱身啟動題5題·5分鐘·複習前置知識
二、核心知識精講3個知識點·3個例題·3個陷阱盒
三、分層練習🌱基礎6題·🌿進階5題·🌳挑戰5題·🏔️極限2題
四、應用題8題SSPA文字題
五、課後功課基礎必做9題·進階選做5題
六、易錯點總結7行易錯表·口訣·自我檢查框
七、解題四步卡4步驟解題框架
家長角落白話文摘要·3大陷阱·檢查問句
教師參考重點題號·答案提示

如何生成答案冊

python scripts/auto_answer_key.py --grade P5 # 只做小五
python scripts/auto_answer_key.py # 全部年級

答案冊自動生成在講義同目錄下，檔名 _AK.html。目前33%題目可自動解答。

👩‍🏫 教學工作流程

上課前

1
打開講義 → 預覽封面資訊（陷阱·目標）
2
檢查「教師參考」區 → 確認本堂重點題號
3
如需要，打開答案冊（_AK.html）參考

上課中（65分鐘流程）

時間環節做什麼
0-5分熱身啟動題學生獨立完成 → 老師快速檢查答案
5-20分核心知識精講講解3個KP → 示範3個例題 → 強調陷阱盒
20-40分課堂分層練習🌱必做 → 🌿選做 → 🌳挑戰。老師巡視指導
40-52分應用題SSPA文字題實戰 → 討論陷阱點
52-60分總結回顧易錯點 → 讀口訣 → 自我檢查
60-65分功課佈置明確指出必做題號 → 預告下堂

下課後（2分鐘內完成）

1
打開「家長報告生成器」 → 填寫學生表現
2
點擊「生成報告」 → 自動複製到剪貼簿
3
WhatsApp 貼上發送給家長
4
（可選）更新「學生進度追蹤器」記錄分數
