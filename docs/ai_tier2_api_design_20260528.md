## 二、Tier 2 功能設計

### 2.1 AI 數學導師 API（核心）

**功能：** 真正的LLM驅動數學輔導

| 特性 | Tier 1（現有） | Tier 2（API） |
|------|-------------|-------------|
| 回答方式 | 預設知識庫匹配 | LLM即時生成 |
| 個性化 | 無 | 根據學生歷史調整 |
| 語言 | 固定繁體中文 | 自然粵語/繁體 |
| 錯誤分析 | 規則型 | 深度理解錯誤思維 |
| 跟進問題 | 無 | 主動追問引導 |

**API 成本估算：**
- 使用 GPT-4o-mini：每1K token ~$0.00015
- 每次對話約500 tokens → $0.000075
- 每日100次對話 → $0.0075/日 → ~$2.7/年/學生
- 167學生 → ~$450/年

### 2.2 AI 智能批改 API

**功能：** 學生上傳答案照片 → AI批改+分析

**技術方案：**
- GPT-4o Vision API（圖片理解）
- 支援手寫數字/算式識別
- 自動判斷對錯+生成詳解

### 2.3 AI 教案生成 API

**功能：** 老師輸入課題 → AI生成完整教案

**輸出內容：**
- 核心概念講解（香港用字）
- 3-5個分級例題
- 陷阱標記（T1-T10分類）
- 互動活動建議
- 課後練習題（5-10題）

### 2.4 AI 語音講解 TTS

**功能：** 廣東話數學概念語音講解

**技術方案：**
- OpenAI TTS（目前不支援廣東話）
- 替代方案：Edge TTS（支援 zh-HK）
- 用途：關鍵概念語音卡、陷阱口訣語音

## 四、安全與成本控制

| 措施 | 說明 |
|------|------|
| API Key 代理 | 不暴露前端，通過 Cloudflare Workers 中轉 |
| 速率限制 | 每學生每日最多20次API調用 |
| 內容過濾 |  prompt injection 防護 |
| 成本預算 | 每月API預算上限 $500 HKD |
| 降級策略 | API不可用時自動切換Tier 1規則引擎 |

## 五、推薦技術棧

| 層級 | 技術 | 原因 |
|------|------|------|
| API 代理 | Cloudflare Workers | 免費額度充足、全球CDN |
| LLM | GPT-4o-mini | 成本低、中文能力強 |
| Vision | GPT-4o | 準確度高（按需使用） |
| TTS | Edge TTS (zh-HK) | 免費、支援廣東話 |
| 監控 | Vercel Analytics | 即時用量追蹤 |
